# 《万族志MUD》玩家能力设计

*——修行之始，在于"觉"。一念之觉，便是凡人与修行者的分界。*

---

## 系统概述

### 设计哲学

> **"天地万物皆浸灵气之中，然凡人不知不觉，如鱼不知水。"**

玩家能力系统不是一条数值阶梯，而是一段**感官被灵气逐层重写**的旅程。每一次升境，世界并非"变得更弱"，而是**在玩家眼中变得更深**——同一片江水，在凡人眼中是鱼汛，在闻道者眼中是脉络，在见界者眼中是与远方某界相通的薄膜。

### 与世界观的关系

- **境界 = 感知层级**：等级数字背后真正递进的是"能看见什么、能听见什么、能触到什么"。
- **力量是副产物**：灵术、伤害、HP上限是感知能力的外显，而非追逐的目标。
- **代价随境界深化**：六境后开始出现明确的副作用——这是世界观"灵气改写人性"的具体兑现。

### 设计原则

1. **易于理解，难于精通**：境界命名、能力解锁线性清晰；但灵术的搭配、灵觉在探索中的应用拥有可挖掘的深度。
2. **成长体验 > 数值膨胀**：每次升境优先解锁**新的世界感知方式**或**新的交互动词**，而非单纯的属性提升。
3. **线性 / 微弱对数增长**：杜绝指数膨胀。Lv.10的玩家面对Lv.5的敌人有压制感但非"一击碾压"，仍需消耗资源。
4. **数值服务于叙事**：若某项数值调整会破坏剧情节奏（例如闻道一境就能秒杀异化江鱼），数值让步。
5. **公式安全**：所有除法采用 `A / (1 + B/K)` 范式，结构上杜绝除零。

## 境界体系完整表格

### 十重境总览（与世界观一致）

| 重境 | 名称 | 等级范围 | 核心主题 |
|------|------|---------|---------|
| — | **凡人** | Lv.0 | 灵气如水，鱼不知水 |
| 第一重 | **闻道** | Lv.1 ~ 9 | "听见"灵气，初识世界另一层 |
| 第二重 | **见界** | Lv.10 ~ 19 | "看见"界壁，知诸界相邻 |
| 第三重 | **通幽** | Lv.20 ~ 29 | "通达"幽微，可与界外低语 |
| 第四重 | **化形** | Lv.30 ~ 39 | 灵气可塑形，借物寄意 |
| 第五重 | **掌缘** | Lv.40 ~ 49 | 掌握因果之缘，牵引微小命运 |
| 第六重 | **逆时** | Lv.50 ~ 59 | 短暂逆转某一刻的"已发生" |

### 小境通用命名

> 每一重大境内，九小境均遵循以下递进节奏与命名。命名取意于"由被动→主动→精纯→显代价→极致"的修行心路。

| 小境序 | 名称 | 状态描述 |
|-------|------|---------|
| 一境 | **初窥** | 能力偶发、不可控，常伴困惑、震颤或恐惧 |
| 二境 | **承引** | 灵气循引而来，仍需借外物（江风、香火、师长引气）触发 |
| 三境 | **凝意** | 可主动凝聚意念调动灵气，但精度有限、易散 |
| 四境 | **驭法** | 能稳定驾驭一两门成形之法，灵术初具雏形 |
| 五境 | **小成** | 能力可稳定输出，作为战斗/探索的可靠手段。该重境"可用"的分水岭 |
| 六境 | **深蕴** | 能力深化、泛化，可灵活变招；微小代价开始浮现（疲倦、感官残留） |
| 七境 | **涉殇** | 力量更进一层，代价显著且**叙事可见**（噩梦、时空错觉、情感淡漠等） |
| 八境 | **临渊** | 已立于此重境的极限，再进一步即触破境之险 |
| 九境 | **圆满** | 此重境之极致表现。突破九境即尝试"破境"入下一重——失败轻则道基受损，重则道心崩碎 |

> 取名说明：**"涉殇"**之"殇"非死亡，而取"未及而损"之意——能力越界，必有所损。**"临渊"**取自"如临深渊"，提示玩家此处突破风险陡增，应做准备。

## 闻道境详细设计（江州篇重点 · Lv.1 ~ 9）

> **总纲**：闻道一重，是凡人初次"听见"灵气的九级阶梯。一境如雏鸟啼鸣，九境如江潮已熟。每升一阶，世界便在玩家耳中、眼中、指尖多出一层从未察觉的纹理。

### 灵气感知渐变机制

闻道境的核心不是"打怪升级"，而是**感官被灵气一层一层打开**。系统在每一小境提供两套场景描写：

- **基础描写**（凡人视角，永远存在）
- **灵觉描写**（境界达到时**叠加显示**，措辞越深越细腻）

| 小境 | 灵气感知状态 | 描写示例（以"江水"为例） |
|------|------------|------------------------|
| Lv.0 凡人 | 无 | "江水浑黄，浪头不高。" |
| Lv.1 初窥 | 偶发闪现，不受控 | "江水浑黄……忽然，你**眼前一花**，仿佛看见水下有什么在缓缓流动。" |
| Lv.2 承引 | 借外物可触发（手浸入水时） | "你将指尖探入江水，**一缕清凉的脉络**沿着指节向上游走。" |
| Lv.3 凝意 | 主动凝神可短暂感知 | "凝神片刻，江水中**纤细的灵脉**如发丝缠绕你的小腿。" |
| Lv.4 驭法 | 短时稳定感知，可粗略分辨灵气浓淡 | "江水的灵脉在你眼中如**水草摇曳**，浓处近黄，淡处近青。" |
| Lv.5 小成 | 持续稳定感知，可"听"灵气流向 | "江水在你耳中**低语成调**——东侧灵脉粗壮，西侧细弱，像两条河在同一床中流。" |
| Lv.6 深蕴 | 可分辨灵气的"成色"与"脏污" | "你听见江水中**夹杂一缕异响**，像金属互相摩擦——有非自然之物正污染此处灵脉。" |
| Lv.7 涉殇 | 感知泛化但**始终在线**，无法关闭 | "你已无法不去听了。市集的喧哗里夹着每一个人微弱的灵气脉动，**像永不停歇的潮水拍打你的鼓膜**。" |
| Lv.8 临渊 | 感知深及"界壁颤动" | "在江水的尽头，你**第一次模糊地看见**——某种薄如蝉翼之物正在轻轻起伏。那是界壁的边缘。" |
| Lv.9 圆满 | 感知触及极限，已等待破境 | "天地之灵气在你周身如潮水汇聚又退去。你知道，再向前一步，便不再是'闻'，而是'见'。" |

### 闻道境能力解锁与战斗对位

> 设计目标：玩家在江州各幕的境界与任务文档中的敌人难度对位。Lv.1-3为公共阶段（所有道途共享），Lv.4-6为道途分化阶段（详见 [《道途设计》](06_道途设计.md)），Lv.7回归公共。

#### 公共阶段（Lv.7-9·全道途统一）

| Lv.8 | 临渊 | 灵术·【界缝窥视】（探索类，可看见界壁裂隙） | HP 170，灵力 150 | 第三幕中段对位 |
| Lv.9 | 圆满 | 准备破境，无新灵术，全属性微调 | HP 195，灵力 180 | 破境试炼·叙事关卡 |

#### Lv.7 副作用：【过载灵敏】（剧情×玩法×数值三方共识）

- **触发**：升至Lv.7自动获得，无法关闭。
- **机制**：
  - 在**喧闹/灵气混杂区域**（市集、码头集会、战斗中受多个灵气源攻击时），HP自然回复速率 ×0.7。
  - 在**安静/灵气清净区域**（清晨江面、深夜竹林、师父小屋），HP/灵力回复速率 ×1.3。
- **叙事兑现**：NPC会注意到玩家"看起来很疲倦"；师父在识海中低语"你已开始听见太多了"（呈现风格见 [《师徒系统设计》§4.3](09_师徒系统设计.md)）。
- **设计意图**：让"代价"成为一种**叙事提示与玩法引导**，而非纯粹的负面属性——玩家会自发寻找清净处休整，呼应剧情的内省节奏。

> **数值校准提示**：上表所有数值为设计中位值，留有 ±15% 调整空间。实测时若发现Lv.5对虫群战斗清场超过8回合，应优先调整【灵气涟漪】的目标数（3→4）而非提升单体伤害，以保护"范围攻击专精"的标识感。

### 灵术觉醒事件机制（自由度·第三层）

设计原则："等级解锁潜能 + 叙事事件点燃觉醒"。灵术不再是升级时自动获得，而是达到等级后，在特定叙事瞬间被"点燃"。每个灵术都有一个"觉醒事件"和一个"保底机制"。

| 灵术 | 等级条件 | 觉醒事件 | 保底机制 | 设计意图 |
|------|---------|---------|---------|---------|
| 灵气弹 | Lv.1 | ★【暗渠之谜】中灵觉初开，灵气自指尖凝出 | 觉醒=自动获得 | 觉醒本身就是叙事高潮，无需额外事件 |
| 灵气微感 | Lv.2 | 首次进入灵气浓郁区域（修辑坊/暗渠深处/镇潮阁附近），灵觉自发"涌入"——"你忽然看到了空气中的光点" | 达到Lv.2后首次移动到任意新房间时自动触发 | 让"感知升级"绑定一个具体的空间体验 |
| 聚气一击 | Lv.3 | 战斗中HP首次低于40%——"危急之中，灵气不受控地猛然聚拢于拳" **或** 在安全环境中师父主动教授（`请教师父 修行`触发） | 达到Lv.3后首次进入战斗自动触发 | 两种路径：战斗型玩家在危机中领悟，探索型玩家由师父点拨 |
| 道途首术 | Lv.4 | ★【道途初分】觉醒事件（到达云栖城，灵气浓度触发灵觉分化） | 到达云栖城后自动触发 | 道途觉醒是不可跳过的叙事节点 |
| 道途Lv.5术 | Lv.5 | 首次使用道途Lv.4灵术击败敌人后，"你感到灵觉更深了一层"——领悟下一阶灵术 | 达到Lv.5后第二次战斗自动触发 | 用上一阶灵术的实战体验"催生"下一阶 |
| 道途Lv.6术 | Lv.6 | 对应Boss战中特定情境触发：观潮/逆流在被灵术击中时顿悟反弹；握锋/灵刃斩在面对非实体时感知到"虚实无别"；鸣钟/钟鸣裂在虫群密集时感到共振频率突破；铸壁/镇守在队友(陶翁)受伤时"必须保护他"的意念催生 | 达到Lv.6后首次Boss战自动触发 | 最高阶灵术的觉醒与Boss战叙事融合 |

> **实现规范**：觉醒事件触发时插入一段沉浸描写（2-3行），风格参照 [《战斗数值设计》](08_战斗数值设计.md) 的战斗描写标准。保底机制确保任何玩家都不会被卡住——自由度的松绑不能变成新的卡点。

## 见界境概要设计（Lv.10 ~ 19 · 第三幕及之后）

> **总纲**：闻道是"听见"，见界是"看见"。九小境均围绕"界壁"展开——从初见界缝，到能短暂触碰界壁，到能在界壁裂处来去。

### 核心能力方向

- **视觉的根本性扩展**：可常态看见界壁（之前只在Lv.8临渊偶现），世界从"一层"变为"多层叠影"。
- **跨界交互**：低阶可隔界听声、传讯；高阶可短暂"借物"——从邻界借一缕风、一片影、一滴水。
- **界壁判读**：可读出界壁的"厚度""年龄""伤口"，成为后续主线"界壁松动"剧情的核心调查能力。

### 预留设计空间（不在江州篇展开）

| 小境 | 命名 | 能力方向（占位） |
|------|------|----------------|
| Lv.10 | 初窥 | 第一次看见界壁，常感眩晕 |
| Lv.12 | 凝意 | **师父境界**——可稳定看见近处界壁（NPC锚点：师父） |
| Lv.15 | 小成 | 可短暂将手探入界壁，借物 |
| Lv.17 | 涉殇 | 双重视觉永久叠加，常分不清此界彼界 |
| Lv.19 | 圆满 | 已可在界壁薄处徒步穿行短距离 |

> 见界境完整设计将在第三幕开发期展开，本章仅占位，确保闻道→见界过渡逻辑顺畅。

## 经验值与升级曲线

### 经验来源占比（沿用任务文档约定）

| 来源 | 占比 | 备注 |
|------|------|------|
| 主线任务 | 40% | 节奏锚点，确保剧情推进≈等级推进 |
| 战斗 | 20% | 可重复但有衰减（同种敌人24h内经验×0.5） |
| 探索 | 15% | 首次到达新区域、首次发现灵脉/灵物 |
| 支线任务 | 15% | 可选叙事内容奖励 |
| 灵觉修行 | 10% | 主动打坐、与师父请教、品鉴灵物 |

### 升级所需经验（江州篇详表）

> **设计原则**：1~10级使用**温和二次曲线**（避免线性带来的"中段乏味"），10级后**每级在前一级基础上+15%**（注：是"递增量"+15%，非总量复合，杜绝指数失控）。

**1~10级公式**（n 为当前等级，求 n→n+1 所需经验）：
```
EXP(n) = round( 100 * (1 + 0.40 * n + 0.10 * n²) )    其中 n ∈ [0, 8]
EXP(9) = 2000   // 9→10为"破境"，叙事性消耗，独立配置
```

**10级后公式**：
```
EXP(n) = round( EXP(n-1) * 1.15 )    其中 n ∈ [10, 上限]
注：1.15 仅在 10~30 级段生效；30 级后将切换为对数封顶曲线（后续幕设计时校准）
```

| 当前等级 | 升至下一级所需 EXP | 累计 EXP | 备注 |
|---------|-------------------|---------|------|
| Lv.0 | 100 | 100 | 第一幕开场觉醒，主线驱动 |
| Lv.1 | 150 | 250 | |
| Lv.2 | 220 | 470 | |

### 数值校验：边际效应检查

> 数值策划论证：以 Lv.4 玩家为例，主线收尾给 600 EXP，单次直接推进至 Lv.5（420需求），剩余 180 EXP 进入 Lv.5。此时玩家**正好踏入第二幕**并立即解锁【灵气涟漪】范围灵术，与"云栖城面对虫群"的剧情节奏精准对位。
>
> 反向校验：若玩家纯刷战斗（Lv.4打Lv.4敌人，48 EXP/次，衰减后24 EXP），需约 17 场战斗才能升级——刷怪疲劳明显，**主线驱动占优**，符合40%主线占比的设计意图。

### 多元经验源（自由度·第三层）

经验不只来自战斗和任务完成，探索/知识/社交都是经验来源。

| 经验类型 | 来源 | 占总经验比建议 | 示例 |
|---------|------|--------------|------|
| 叙事经验 | 推进主线/支线里程碑 | 50-60% | 完成【暗渠之谜】、推动主线关键剧情 |
| 探索经验 | 首次发现新房间/新区域 | 15-20% | 首次进入镇潮阁（+30 EXP）、首次发现暗渠深处（+20 EXP） |
| 知识经验 | 获得新信息片段 | 10-15% | 从洪大爷处获得[古钟来历]（+15 EXP）、从旧书获得同信息（+10 EXP，浅版经验较少） |
| 战斗经验 | 击败敌人 | 10-15% | 击败异化江鱼（+10 EXP） |
| 社交经验 | NPC好感度里程碑 | 5-10% | 阿寿好感度达到"信赖"（+20 EXP） |

> 1. **和平主义可行**：一个完全不战斗的玩家，通过探索+知识+社交也能升到Lv.3-4，足以应对第一幕的叙事需求。
> 2. **信息深浅影响经验**：从"深度来源"（如洪大爷详细讲述）获得的信息经验高于"浅度来源"（如旧书简要记载），鼓励深度对话。
> 3. **首次发现奖励**：每个房间/信息只在首次发现时给予经验，不可重复。
> 4. **数值校准提示**：上表比例为设计目标，需在实测中调整。核心约束：纯战斗路线和纯探索路线的升级速度差异不超过±30%。

## 灵术系统

### 灵术分类

| 类别 | 定位 | 核心机制 |
|------|------|---------|
| **攻击** | 主要伤害手段 | 消耗灵力，造成单体或范围伤害 |
| **防御** | 减伤、护盾、规避 | 主动开启，按回合或按吸收量结算 |
| **感知** | 探索、识别、预判 | 战斗外为主，部分战斗内可"先手揭示" |
| **辅助** | 增益、回复、控制 | 自身或友方加成，少量控制效果 |

### 江州篇灵术总表

> **道途说明**：Lv.1-3 为公共灵术（所有道途共享），Lv.4-6 为道途分化灵术（按所选道途解锁），Lv.7+ 回归公共。详见 [《道途设计》§四](06_道途设计.md)。

#### 公共灵术（全道途共享）

| 灵术名称 | 类别 | 解锁等级 | 灵力消耗 | 效果 | 设计意图 |
|---------|------|---------|---------|------|---------|
| **灵气弹** | 攻击 | Lv.1（觉醒） | 10 | 单体伤害 8~12 | 修行第一术，象征"凡人→修行者"的分界 |
| **灵气微感** | 感知（被动） | Lv.2 | 0 | 探索时自动揭示3格内微小灵物 | 鼓励探索，呼应"承引" |
| **聚气一击** | 攻击 | Lv.3 | 15 | 蓄力1回合，下次普攻+12伤害 | 引入回合策略 |
| **灵气共鸣** | 辅助 | Lv.7 | 维持费10/回合 | 可同时维持两个其他灵术（如盾+蓄力） | 体现"涉殇"——能力深化但持续耗费 |
| **界缝窥视** | 感知 | Lv.8 | 40（一次性） | 揭示当前场景内所有界壁裂隙与隐藏出口 | 为见界境铺垫 |

#### 道途灵术（Lv.4-6·按所选道途解锁其中一列）

| 等级 | 观潮（控制/辅助） | 握锋（单体输出） | 鸣钟（范围输出） | 铸壁（防御/承伤） |
|------|----------------|----------------|----------------|-----------------|
| Lv.4 | **灵流锁**（控制·消18）定身1目标2回合 | **穿灵刺**（攻击·消15）单体18-22，无视20%DEF | **灵气涟漪**（攻击范围·消25）范围6-10/单位，≤3目标 | **凝水盾**（防御·消20）护盾值25，持续3回合 |
| Lv.5 | **潮引术**（控制·消22）改变敌人攻击目标 | **破隙斩**（攻击·消22）单体22-28，蓄力目标×1.5 | **共振波**（攻击范围·消28）范围8-12，≥3目标时+3/个 | **灵壁术**（防御·消25）范围护盾，覆盖自身+1友方 |

> **灵术类型与敌人机制交互**：道途灵术分为纯灵术型、锋刃型、特化型、防御型四类，与敌人特殊机制（非实体减伤、灵气压制光环等）有不同交互规则——详见 [《战斗数值设计》§数值平衡验证·交互规则表](08_战斗数值设计.md)。

### 灵术升级机制

每个灵术拥有 **熟练度（0~100）**，通过使用累积：

- **熟练度 0~30**：基础形态。
- **熟练度 31~70**：**精炼**。灵力消耗 -15%，效果数值 +10%。
- **熟练度 71~100**：**圆融**。解锁该灵术的小变体（例：灵气弹·圆融可附带"短暂揭示敌人灵气成色"的副效果）。

> 单次战斗中同一灵术使用上限计入熟练度的次数 = 3，防止刷熟练度。

### 灵力消耗与恢复

- **灵力上限** = 20 + 等级 × 15（粗算公式，留 ±10% 调整空间）
- **战斗外恢复**：每分钟（或每次行动）恢复 上限的 10%
- **战斗内恢复**：每回合恢复 上限的 5%
- **打坐**：30秒（现实时间）恢复 100%灵力 + 10 EXP，每场打坐受冷却限制
- **特殊恢复**：吴婶的鱼丸汤、清净灵脉处冥想可额外加成（呼应NPC设计）

#### 战斗伤害公式（防溢出范本）

```
最终伤害 = 基础伤害 * 50 / (50 + 目标防御)
说明：50 为缓冲常数 K。即使目标防御为 0，分母仍为 50，结构上不可能除零。
当目标防御为 50 时，伤害减半；防御 150 时，伤害降至 1/4。
```

> 数值策划注：所有除法运算均采用 `A / (1 + B/K)` 或 `A * K / (K + B)` 范式，K 为正常数（本作约定 K=50），结构上杜绝除零。

## 灵觉感知系统

### 灵觉等级与境界对应

> 灵觉是"感知灵气"的能力，它**总是与境界强相关，但可独立修行而略有领先或滞后**。如赵船主、吴婶虽未正式修行，长年浸润灵气也已属闻道一境的灵觉萌芽状态。

| 灵觉等级 | 对应境界 | 描述 | 典型NPC |
|---------|---------|------|---------|
| 灵觉萌芽 | 凡人（部分） | 仅有"直觉""第六感"，本人无察 | 赵船主、郑掌柜、吴婶 |
| 灵觉初开 | 闻道一~二境 | 偶能察觉灵气流动，常被误以为眩晕 | 陈老鱼头、鱼牙子老周 |
| 灵觉成形 | 闻道三~四境 | 可主动调用灵觉做粗略判断 | 葛师傅、鲁伯 |
| 灵觉稳定 | 闻道五~六境 | 持续在线，可分辨灵气浓淡成色 | 苦力阿山 |
| 灵觉深化 | 闻道七~九境 | 感知泛化、永不关闭，**伴随过载副作用** | 素娥（圆满） |
| 灵觉跨界 | 见界一境及以上 | 可见界壁，感知跨越单一界域 | 师父（见界三境） |

### 觉醒前后的世界描写差异（双版本机制）

系统对场景描写采用**叠加式**：

```
凡人版（基础）：
  你站在江州鱼埠的码头上。江水浑黄，浪头不高。一艘旧船拴在桩上，
  船板被晒得发白。

闻道一境版（叠加）：
  你站在江州鱼埠的码头上。江水浑黄，浪头不高——但在你新生的灵觉中，
  江面之下有【一缕极淡的青光】缓慢游动，像被江水裹着的一条无形之鱼。
  那艘旧船的桩柱根部，似乎也有【极细的灵气脉络】渗进木头的纹理。

闻道五境版（叠加+成色）：
  ……江水之下的青光已成【数股交织的脉络】，东侧粗壮，西侧细弱。
  船桩根部的灵气脉络呈现【清而不润】之色——这艘船的木料采自上游
  灵气稀薄之地。
```

> 实现建议：每个房间/物体在数据中存储多套描写（基础 / Lv.1+ / Lv.5+ / Lv.7+），由系统按玩家当前境界自动叠加输出。

### 灵觉在探索中的应用

| 应用场景 | 触发境界 | 收益 |
|---------|---------|------|
| **隐藏物品发现** | Lv.2+ | 房间描述中以【】标注微小灵物，可拾取 |
| **灵物等级识别** | Lv.3+ | 物品/食材的实际灵气等级显示，否则只显示"未知" |
| **危险感知** | Lv.4+ | 进入有强敌或灵气异常区域时，给出"心头一沉"等提示 |
| **灵脉判读** | Lv.5+ | 可识别灵脉走向、强弱，找到打坐效率更高的位置 |
| **NPC境界识别** | Lv.6+ | 对话/查看NPC时，显示其灵气成色，可推测其大致境界 |
| **界壁裂隙感知** | Lv.8+ | 接近界壁薄处时获得提示，为见界境与第三幕铺垫 |

### 灵觉修行（独立成长线）

灵觉除随境界自动提升外，可通过以下方式**单独深化**（最多领先境界 0.5 级的感知精度）：

- **打坐**：每日上限 100 EXP，灵觉熟练度 +1/次
- **品鉴**：与郑掌柜评鱼、与鱼牙子老周辨灵气成色，每次 +2 灵觉熟练度
- **师父点拨**：每幕 1~2 次，每次 +10 灵觉熟练度（第一幕为实体引导；第二幕起为识海低语，效果不变但呈现形式改为内心声音）

> 设计意图：让"凡人NPC如吴婶亦可达到闻道一境灵觉"这一设定有玩法对应——玩家亦可通过"生活化"修行加深感知，而非只能砍怪练级。

### 残魂师父对玩家修行的指导方式

> 详见 [《师徒系统设计》](09_师徒系统设计.md)

自镇潮阁师父化为残魂后，师父对玩家修行的引导方式从"身旁点拨"转为"识海低语"。核心变化如下：

| 阶段 | 引导形式 | 触发条件 | 示例 |
|------|---------|---------|------|
| **第一幕**（Lv.0~4） | 实体对话 | 师父主动开口或玩家 `talk 师父` | "闭上眼。不要听。不要想。只是——感觉。" |
| **第二幕**（Lv.4~7） | 识海低语 | 师父在关键节点自动触发，或玩家 `请教师父`/`am` | *识海中传来一个声音："这座城的灵气……不对劲。你自己感受。"* |
| **第三幕**（Lv.7~10+） | 偶尔回应 | 仅在玩家请教时小概率回应，主动触发极少 | *很久之后，识海深处传来一丝微弱的回响："……你已经知道答案了。"* |

**设计红线**：
- 残魂师父**绝不提供战斗干预**——无增伤、无减伤、无复活。
- 师父是"百科全书"而非"攻略书"——可解释世界观、灵气知识、NPC背景，但**不直接给出任务答案**。
- 声音频率随玩家境界提升而递减——"从密到疏的消散曲线"是刻意的叙事设计，不是bug。

> **结语**
>
> 闻道一境的玩家，第一次"听见"江水之下灵脉的低语时，他听见的不仅是灵气——他听见的是这个世界**第一次愿意对他说话**。
>
> 此后每一次升境，都是世界更深一层的回应。直到有一天，他听得太多，不得不学会闭上一只耳朵——那便是涉殇。
>
> 而再之后，他将看见界壁。
>
> *——浮黎洲的修行，从来不是关于变强，而是关于"听见多少、看见多少、又愿意承受多少"。*
