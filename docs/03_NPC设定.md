# 《万族志MUD》NPC设定

*——每一个NPC都有自己在乎的事*

---

## 设定总则

1. **NPC不是任务机器**：每个NPC都有独立的动机、日常作息和情感，玩家可以与之闲聊而非只触发任务。
2. **对话反映身份**：渔民说渔谚，商贾算账目，修士引经据典。年龄、阅历、性格都应体现在遣词用句中。
3. **时间敏感**：部分NPC在不同时段出现在不同地点，或展现不同状态（如陈老鱼头清晨在码头，午后在竹棚下）。
4. **关系网络**：NPC之间存在师徒、亲属、同行、交易等关系，玩家可通过一个NPC得知另一个NPC的信息。
5. **隐藏深度**：多数NPC拥有不会主动提及的秘密或过往，需要特定条件（好感度、物品、多轮对话）才会透露。

---

# 零、核心NPC

> 师父是唯一跨越全部幕次的核心NPC，不隶属于任何单一地点。详见 [《师徒系统设计》](09_师徒系统设计.md)。

## 师父（无名）

| 属性 | 描述 |
|------|------|
| 所在 | 第一幕：随玩家同行（鱼埠各处）；第一幕末：镇潮阁三层；第二幕起：玩家识海 |
| 年龄 | 外表中年（实际年龄不明） |
| 身份 | 玩家的师父，引路人 |
| 境界 | 见界三境（Lv.12） |

**外貌**：中年人，布衣芒鞋，腰间别一支青竹杖。面容普通但眼神温和，望向远处时有种"看穿了什么"的疲惫。手指修长有茧——竹杖握久了的人特有的掌纹。头发半白，但不是苍老那种白，更像是灵气侵蚀后的褪色。

**性格**：话少但句句有分量，习惯用行动而非语言守护。偶尔幽默，面对危险异常冷静。骨子里有深沉的疲倦——但从不让徒弟看见这一面。

**背景**：来历不明。玩家不知道师父从哪里来、为什么收自己为徒、为什么执意前往江州。他偶尔提起"老朋友""很久以前的事"，但从不展开。唯一确定的是他对浮黎洲的了解远超一般见界修士——灵脉走向、万族掌故、各地禁忌，他似乎都知道。

**日常行为**（第一幕·实体阶段）：
- 清晨在江边站桩，闭目感知灵脉；白天沉默跟随玩家，关键时刻才开口
- 傍晚独自坐在某处高地（如栖浪酒家二楼窗边），望向远方
- 玩家睡着后，在周围布下微弱的灵气结界（凡人看不见）

**对话风格**：
- 极简，从不直接给答案——"你自己看。""闭上眼，用感觉。""想想为什么。"
- 点到为止——在玩家即将理解时收声，让顿悟由玩家自己完成
- 极少表达情感，但在镇潮阁牺牲前的最后一段话打破了所有克制

**游戏功能**：
- 第一幕：保护型NPC（战斗中可触发保护机制）、觉醒引导者、世界观知识来源
- 第二幕起：识海顾问（`请教师父`/`am` 指令）、世界观百科全书、叙事伴侣
- 未来AI接入：可升级为基于知识库的对话AI，保持人设一致性

**关键关系**：
- 与**玩家**：一生中唯一的徒弟。从不说"我爱你"之类的话，但所有行为都指向一个方向——让这个孩子能独自走下去
- 与**洪氏先祖**：似乎知道镇潮阁的秘密。洪氏先祖残魂在他灵识融入玩家识海后向他致礼，说明他们有某种跨越时间的联系
- 与**陆清玄**：通玄门守门修士。师父曾托陆清玄转达一句话——"让他来找你"。（此条在第二幕由陆清玄触发隐藏对话揭示）

**隐藏触发**：
- 第一幕中，若玩家多次`look 师父`，会累积不同的观察描写——从"一个沉默的中年人"到"你第一次注意到他鬓角的白发"到"他在看你。目光里有一种你读不懂的东西"
- 觉醒后再`look 师父`，描写增加灵气维度——"师父身上的灵气成色极深极稳，如一潭不见底的深水"

### 残魂阶段特别设定

师父在镇潮阁牺牲后，以残魂形态存在于玩家识海中。此阶段不再是传统NPC，而是一种**内嵌于玩家自身的交互实体**。

- **呈现方式**：不出现在房间NPC列表中。通过 `请教师父`/`am` 指令或自动触发呈现
- **声音渐稀**：第二幕初期回复频率高，中期降低，后期仅偶尔回应。这是刻意的"消散曲线"设计
- **知识范围**：世界观、修行知识、各地掌故、NPC背景——但**不提供任务攻略答案**
- **零战斗能力**：不可提供任何战斗加成，详见 [《战斗数值设计》](08_战斗数值设计.md)

## 渔户营

### 陈老鱼头

| 属性 | 描述 |
|------|------|
| 所在 | 渔户营 |
| 年龄 | 七十余岁 |
| 身份 | 渔户营最年长的老渔民，非官方"村长" |
| 境界 | 闻道二境（Lv.2）——七十年江上生涯，灵觉已开，未曾正式修行 |

**外貌/对话**：满脸风霜如老树皮，花白头发草绳束着，穿破旧但干净的粗布短衫。嗓门洪亮，说话爱拍大腿，爱用渔谚——"三月鱼肥四月虾，五月螃蟹往上爬"。

**性格/背景**：热心、健谈、护短，念旧。十三岁随父出江，经历三次百年大潮。妻子洪水中失踪后将渔户营视为自己的家。年轻时曾在镇潮阁附近听见水底钟鸣——若玩家携带相关物品追问，他会缓缓道来。

**日常行为**：清晨在码头边检查渔网绳结，午后坐在竹棚阴凉处给晚辈讲掌故，傍晚独自站在栈道尽头望江，嘴里嚼着干鱼丝。

**游戏功能**：主线引导NPC。提供鱼埠背景信息与风土人情，触发【鱼埠初至】主线。

**关键关系**：
- 与赵船主是四十年老邻居，看不惯他吹牛但暗地里关心。与洪大爷年轻时一起出过海，知道洪家与镇潮阁的渊源但守口如瓶。
- 视阿鲤如孙女，常叮嘱她别跑太远。

|------|------|

**游戏功能**：触发【失踪的渔船】支线入口。任务完成后提供感谢奖励，并成为后续商路支线的潜在NPC。
