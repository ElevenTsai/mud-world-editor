## <a id="验证1"></a>验证 1：Lv.1 玩家 vs 异化江鱼 ×2（Lv.2）

**玩家** Lv.1：HP 50，MP 30，ATK 10，DEF 7，SpA 3，SpR 2，SPD 11
**敌方** 异化江鱼 ×2 Lv.2：HP 25（已因阶段二交锋降低至约 **18**），ATK 7，DEF 5，SpR 3，SPD 14（先手）

> **叙事前提**：主线三"暗渠之谜"阶段二中，玩家以凡人之身战异化江鱼至必败，师父出手。此时江鱼已受师父灵气震荡波及，HP 从满血降至约 18/25（72%）。玩家觉醒后以 Lv.1 满状态（HP 50）重新对战残血江鱼。

**伤害预算**：
- 玩家普攻 → 鱼：`10 × 50/55 × 0.95`(等级差-1) ≈ **8.6**
- 玩家灵气弹 → 鱼：`10 × 1.03 × 50/53 × 0.95` ≈ **9.2**
- 鱼普攻 → 玩家：`7 × 50/57 × 1.06`(等级差+1对鱼) ≈ **6.5**

**回合推演**（聚焦优先击杀A鱼）：

| 回合 | 玩家行动 | 玩家HP/MP | 鱼A HP | 鱼B HP |
|-----|---------|----------|--------|--------|
| T1 | 灵气弹A（消10）| 50 / 20 | 18→8.8 | 18 |
|    | 鱼A咬：-6.5 / 鱼B咬：-6.5 | 37.0 / 20 | | |
| T2 | 普攻A → **击杀** | 37.0 / 20 | 死 | 18 |
|    | 鱼B-6.5 | 30.5 / 20 | | |
| T3 | 灵气弹B（消10）| 30.5 / 10 | — | 18→8.8 |
|    | 鱼B-6.5 | 24.0 / 10 | | |
| T4 | 普攻B → **击杀** | 24.0 / 10 | — | 死 |

**结论**：4 回合险胜，HP 余 24/50（48%，轻伤区间）。鱼的残血设定确保"觉醒后力量爆发"的正反差体验——凡人态打不动的敌人，灵术一发即伤。✓

> **剧情策划交叉确认**：战后 HP 约半血，仍足以让师父说出"不错，但别大意"而非"你差点死了"——与阶段二（必败）的恐惧形成层次分明的情绪递进：**绝望 → 觉醒 → 初胜 → 余悸犹在**。师父灵气疗愈手补满HP，开启后续探索。

## <a id="验证4"></a>验证 4：Lv.5 玩家 vs 灵气异化虫群（Lv.5，6只·保护陶翁）

> **道途影响**：虫群战发生在第二幕，玩家已选择道途。下面分道途验证。

**玩家** Lv.5：HP 110, MP 85, ATK 18, DEF 15, SpA 15, SpR 10, SPD 15
**虫群** ×6：每只 HP 9, ATK 5, DEF 2, SpR 2, SPD 13
**陶翁**（NPC）：HP 50，需存活至战斗结束
**虫群 AI**：优先攻击陶翁（约 60% 目标为陶翁，40% 玩家）；每回合末从环境补充 1 只（直至灵气异化源被净化）

> **设计要点**：虫群持续补充，纯单杀无法终结——须**破坏灵气异化源**方可净化环境。各道途的"净化"路径不同：
> - **鸣钟**：灵气涟漪/共振波的范围冲击波直接粉碎灵气异化源（★最高效）
> - **观潮**：潮引术引导灵气回流，逆向堵塞异化源（3 回合引导完成）
> - **握锋**：灵觉探查灵气裂缝位置 → 穿灵刺精准击破异化源（操作挑战）
> - **铸壁**：灵壁术封锁异化源溢出口，持续 3 回合后源头窒息

#### 路线 A·鸣钟（★最契合）

**伤害预算**：
- 灵气涟漪（消25，3目标）：`8 × 1.15 × 50/52` ≈ **8.85/只**（虫HP 9，一击杀）
- 共振波（消28，净化冲击）：对灵气异化源造成结构性破坏

> **数值修正**：虫群单只 HP 沿用修正值 9（原设计 12，因涟漪清场效率不达标已调低）。

| 回合 | 玩家行动 | 玩家HP/MP | 虫群 | 陶翁HP | 备注 |
|-----|---------|----------|------|-------|------|
| T1 | 灵气涟漪（消25）→ 清 3 只 | 110 / 60 | 6→3 | 50 | |
|    | 2 虫攻陶翁 -7.3，1 虫攻玩家 -3.7 | 106.3 / 60 | +1=4 | 42.7 | |
| T2 | 灵气涟漪（消25）→ 清 3 只 | 106.3 / 35 | 4→1 | 42.7 | |
|    | 1 虫攻陶翁 -3.7 | 106.3 / 35 | +1=2 | 39.0 | |
| T3 | **共振波**（消28）→ 清残虫 + **灵气源净化** | 106.3 / 7 | 0 | 39.0 | **环境净化！不再补充** |

**结论**：3 回合清场 + 净化，HP 106/110（几乎无伤），灵力余 7/85。陶翁 HP 39/50（安全）。鸣钟的范围清扫天然克制群体战。✓

#### 路线 B·握锋（最不契合）

**伤害预算**：
- 普攻：17.3（一击杀 1 只虫，但每回合仅杀 1 只）
- 穿灵刺（消15）：对虫溢出伤害严重，用于击破灵气异化源

| 回合 | 玩家行动 | 玩家HP/MP | 虫群 | 陶翁HP | 备注 |
|-----|---------|----------|------|-------|------|
| T1 | 普攻杀 1 虫 | 110 / 85 | 6→5 | 50 | |
|    | 3 虫攻陶翁 -11.0，2 虫攻玩家 -7.3 | 102.7 / 85 | +1=6 | 39.0 | |
| T2 | 普攻杀 1 | 102.7 / 85 | 6→5 | 39.0 | |
|    | 3 虫攻陶翁 -11.0，2 虫攻玩家 -7.3 | 95.4 / 85 | +1=6 | 28.0 | |
| T3 | **灵觉探查**（操作挑战，消15）→ 定位灵气裂缝 | 95.4 / 70 | 6 | 28.0 | 用穿灵刺精准感知裂缝 |
|    | 3 虫攻陶翁 -11.0，3 虫攻玩家 -11.0 | 84.4 / 70 | +1=7 | 17.0 | ⚠️ 陶翁告急 |
| T4 | **穿灵刺击破裂缝**（消15）→ **灵气源被切断！** | 84.4 / 55 | 7 | 17.0 | **净化！不再补充** |
|    | 4 虫攻陶翁 -14.6，3 虫攻玩家 -11.0 | 73.4 / 55 | 7 | **2.4** | ⚠️ 陶翁濒危！ |

**结论**：7 回合，HP 余 ~50/110（45%）。**陶翁仅剩 2.4 HP**，极度危险。若 T3 探查失败或操作延迟 1 回合则陶翁必死——完美体现"握锋不擅长保护战"的设计差异。✓

> **数值策划备注**：
> - 陶翁 HP 设定为 50：鸣钟路线游刃有余（陶翁余 39），铸壁路线陶翁全程安全（灵壁术直接覆盖），观潮路线稳健（潮引术改变虫群目标），握锋路线惊险过关（陶翁余 2.4）。
> - **握锋玩家的提示**：若灵觉探查成功，系统会在 T3 给出"你感觉到裂缝在某处脉动"的文字引导，暗示玩家下一回合应优先击破源头而非恋战杀虫。

## <a id="验证4-5"></a>验证 4.5（新增）：Lv.5·握锋 vs 齿轮傀儡（Lv.5·引航台）

> **道途影响**：齿轮傀儡的核心机制是"鳞片灵气压制"光环（5 格内灵术伤害 ×0.5）。握锋的"锋刃化灵术"以灵气凝形为物理刃，受光环压制较弱（×0.75 而非 ×0.5）；加上穿灵刺无视部分 DEF + 破隙斩抓重击前摇，握锋为此 Boss 的**最契合道途**。

**玩家** Lv.5·握锋：HP 110, MP 85, ATK 18, DEF 15, SpA 15, SpR 10, SPD 15
**齿轮傀儡** Lv.5：HP 70, ATK 16, DEF 14, SpR 8, SPD 17（先手）
- **鳞片灵气压制**光环：5 格内灵术伤害 ×0.5（握锋锋刃型 ×0.75）
- **行为循环**：轻 → 轻 → 重（"重"前 1 回合齿轮加速声预警）

**伤害预算**：
- 普攻：`18 × 50/64` ≈ **14.1**（不受光环影响）
- 穿灵刺（消15，ignore 20% DEF，光环 ×0.75）：`20 × 1.15 × 50/(50+14×0.8) × 0.75` ≈ **14.1**
- 破隙斩（消22，对蓄力 ×1.5，光环 ×0.75）：`25 × 1.15 × 50/61.2 × 0.75` ≈ **17.6**；蓄力时 → **26.4**
- 灵气弹（光环 ×0.5）：`10 × 1.15 × 50/58 × 0.5` ≈ **5.0**（被严重压制，不推荐）
- 傀儡轻攻：`16 × 50/65` ≈ **12.3**
- 傀儡重攻（×1.3）：`16 × 1.3 × 50/65` ≈ **16.0**

| 回合 | 玩家行动 | 玩家HP/MP | 傀儡HP | 备注 |
|-----|---------|----------|-------|-----|
| T1 | 穿灵刺 → -14.1（消15）| 110 / 70 | 55.9 | 傀儡先手·轻攻 -12.3 → HP 97.7 |
| T2 | 穿灵刺 → -14.1（消15）| 97.7 / 55 | 41.8 | 傀儡·轻攻 -12.3 → HP 85.4；**预警：齿轮加速！** |
| T3 | **破隙斩**（抓蓄力 ×1.5）→ -26.4（消22）| 85.4 / 33 | 15.4 | 傀儡·重攻 -16.0 → HP 69.4 |
| T4 | 穿灵刺 → -14.1（消15）| 69.4 / 18 | 1.3 | 傀儡·轻攻 -12.3 → HP 57.1 |
| T5 | 普攻 → -14.1 → **击杀** | 57.1 / 18 | 死 | 傀儡·轻攻… 已死 ✓ |

**结论**：5 回合，HP 余 57.1/110（52%，**轻伤**），灵力余 18/85。握锋通过"锋刃型灵术抗压制 + 破隙斩抓蓄力窗口"高效击杀，符合策略矩阵 5-6 回合预估。✓

> **对比**：鸣钟路线灵气涟漪被光环压制 ×0.5，需 7-8 回合；铸壁路线灵壁术隔绝光环后稳定输出，需 8-9 回合但几乎无风险。握锋最快但吃重攻 16.0 最疼——"高效但需精准操作"的道途体验。

## <a id="验证5"></a>验证 5：Lv.6·握锋 vs 水兽触须幻影（Lv.6，中Boss·非实体）

> **道途影响**：触须幻影为非实体，物理伤害 ×0.5。握锋·灵刃斩的核心特性"对非实体无衰减"正是为此类敌人设计，故握锋为**最契合道途**。握锋其余锋刃型灵术（穿灵刺/破隙斩）因含物理成分仍受 ×0.5 衰减。

**玩家** Lv.6·握锋：HP 130, MP 110, ATK 20, DEF 17, SpA 18, SpR 12, SPD 16（先手）
**触须幻影** Lv.6：HP 100, ATK 22, DEF 8（**物理 ×0.5**），SpA 18, SpR 14, SPD 12

**伤害预算**：
- 普攻（×0.5 非实体）：`20 × 50/58 × 0.5` ≈ **8.6**（极低）
- 灵气弹（纯灵术，不受非实体减伤）：`10 × 1.18 × 50/64` ≈ **9.2**（消10）
- **灵刃斩**（★无衰减）：`21 × 1.18 × 1.0` ≈ **24.8**（消30）
- 穿灵刺（锋刃型 ×0.5 非实体）：`20 × 1.18 × 50/56.4 × 0.5` ≈ **10.5**（消15）
- 触须普攻：`22 × 50/67` ≈ **16.4**
- 触须横扫（每 4 回合周期）：击退，玩家跳过下回合行动

#### 推演 A·无闪避

| 回合 | 玩家行动 | 玩家HP/MP | Boss HP | 备注 |
|-----|---------|----------|--------|-----|
| T1 | 灵刃斩 → -24.8（消30）| 130 / 80 | 75.2 | 玩家先手 |
|    | 触须普攻：-16.4 | 113.6 / 80 | | |
| T2 | 灵刃斩 → -24.8（消30）| 113.6 / 50 | 50.4 | **HP 50% 转阶段** |
|    | 触须普攻：-16.4 | 97.2 / 50 | | |
| T3 | 灵刃斩 → -24.8（消30）| 97.2 / 20 | 25.6 | |
|    | 触须横扫：击退 | 97.2 / 20 | | 玩家跳过 T4 |
| T4 |（击退跳过）| 97.2 / 25 | 25.6 | 回灵 +5 |
|    | 触须普攻：-16.4 | 80.8 / 25 | | |
