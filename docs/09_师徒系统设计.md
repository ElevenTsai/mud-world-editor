# 《万族志MUD》师徒系统设计

*——师父不是NPC，是世界送给玩家的第一盏灯*

---

## 系统概述

### 设计哲学

师父之于玩家，是世界递到手中的**第一盏灯**：照亮觉醒前的三尺路，在玩家能自己点灯时悄然熄灭。他的整个生命周期——从实体陪伴到识海低语，从话多到沉默——本身就是一段叙事：我们不"杀死"师父，我们让他**慢慢远去**。

### 叙事定位

师徒关系是江州篇的**情感主线**，与"觉醒—成长—独立"的能力主线相互交织：

| 阶段 | 能力主线 | 师徒情感主线 |
|------|----------|--------------|
| 第一幕（鱼埠 Lv.0→4） | 凡人 → 闻道四境，灵觉初开 | 实体陪伴，从依赖到信任 |
| **过渡（镇潮阁高潮）** | **觉醒后第一次独自承担重任** | **师父化为残魂，融入识海** |
| 第二幕（云栖城 Lv.4→7） | 闻道四境 → 闻道七境 | 识海低语，从倾听到独行 |
| 第三幕及以后 | 闻道七境 → 触及见界 | 残魂渐稀，最终归于沉默 |

### 两种形态概述

```
   ┌──────────────────────────────────────────────────┐
   │                                                  │
   │   实体阶段（第一幕）          残魂阶段（第二幕起）  │
   │   ─────────────────           ──────────────────  │
   │   ◆ 见界三境修行者            ◆ 一缕灵识          │
   │   ◆ 同行可见                  ◆ 寄于玩家识海      │
   │   ◆ NPC直接互动               ◆ "内心声音"        │
   │   ◆ 危急时可出手保护          ◆ 完全无法干预战斗  │
   │   ◆ 引导玩家觉醒              ◆ 提供信息与解读    │
   │                                                  │
   │             ↓ 镇潮阁·叩钟镇潮 ↓                  │
   │       （肉身崩溃，灵识碎裂入识海）                │
   │                                                  │
   └──────────────────────────────────────────────────┘
```

## 第一幕：实体师父

### 基础设定

> 师父完整人设见 [《NPC设定》§师父](03_NPC设定.md)。此处仅记录师徒系统专属设计。

- **称谓**：师父（玩家可在序章自定义其名号，默认"无名"）
- **流派**：偏感知/灵觉一脉的修行者，非战斗系强者

**境界定位**：见界三境（Lv.12）相对主要敌人（Lv.1~Lv.7）形成压制，面对水兽（Lv.66）毫无胜算——闭合"师父为什么不直接帮打"的叙事逻辑。

#### 为什么带徒弟来江州

这是一个看似闲笔、实则贯穿三幕的伏笔。师父的真实目的应当**对玩家全程隐藏**，仅通过零散对话与觉醒后回望露出冰山一角：

- **表面理由**（任何NPC问起时师父的回答）：*"带徒弟来看看烟火气。"*
- **真实理由**（设计层面，玩家需在第三幕方能完全拼出）：师父早已感知到江州一带界壁松动、灵脉北延、镇潮阁封印将弛的征兆。他带徒弟来此，**既是为了让徒弟在最贴近界壁松动处觉醒**（觉醒效率最高），**也是为了——若万一封印崩溃——以见界三境之力为后续修行者们争取时间**。

#### 灵觉感知

师父的见界三境意味着他能**稳定看见近处界壁**、**清晰感知一切灵气流动**。这造成了第一幕中一个动人的张力——

**师父看到的世界**与**玩家看到的世界**截然不同：

| 场景 | 凡人玩家所见 | 师父（暗中）所见 |
|------|--------------|------------------|
| 鱼埠码头 | 渔民吆喝、鱼腥味 | 江面下方有一缕异色灵气逆流 |
| 暗渠入口 | 阴湿的石壁，腥气 | 灵气在此处出现明显裂痕 |
| 镇潮阁远景 | 一座孤岛上的旧阁 | 整座阁笼罩在三百年前的镇压法阵残光中 |
| 阿旺的眼睛 | 一个普通学徒少年 | （没有特殊——但师父会多看一眼，因为他知道每个少年都可能是某个未来的"自己"） |

### 行为模式

#### 跟随机制

实体阶段的师父采用**跟随但不主导**的设计：

- 师父始终位于玩家身后一步的位置（场景描写中固定提及"师父立于身后"）
- NPC的对话默认面向玩家，但**关键NPC（葛师傅、阿寿、孙老六等）会先与师父点头示意，再与玩家说话**——这是江湖辈分的体现
- 师父**不参与玩家的物品栏、不占用队伍格、不消耗资源**——他是叙事存在，不是数值存在
- 玩家可输入 `look master` / `观察师父` 查看当前师父的神情、动作（动态描写，与场景挂钩）

#### "师父在场"对场景的渲染

第一幕几乎所有场景描写都需预留**师父侧栏**——一段简短的、独立于主场景描写的师父细节，觉醒后回看会"刷新"。凡人版与觉醒版场景正文差异仅在灵气维度，而师父侧栏文字本身不变——变的只是玩家的理解。完整演出示例见 [《江州篇剧本》](10_江州篇剧本.md)。

#### 渐进式对话

师父在第一幕的对话风格遵循**从生活化到点拨**的渐进：

| 阶段 | 对话风格 | 示例 |
|------|----------|------|
| 初入鱼埠（Lv.0） | 生活化、近乎闲谈 | *"先尝尝阿婆的鱼羹，江州的水养出的鱼，骨头都是甜的。"* |
| 接触异常（Lv.1之前） | 含混、回避正面回答 | *"江水的事，你先别想太多。看眼前。"* |
| 觉醒前夜 | 凝重、欲言又止 | *"今晚这场雨，你会记一辈子的。"* |
| 觉醒之后（Lv.1+） | 简洁、像在跟同行者说话 | *"你现在看见的，比昨天多了，也比明天少。"* |
| 镇潮阁前夕（Lv.3） | 异常温柔、像在道别 | *"无论钟响之后发生什么，记住——闭上眼，去感觉。"* |

### 觉醒场景中的角色（保留与精修）

主线三【暗渠之谜·灵觉初开】保留现有设计的核心，本节不做覆盖式修改，仅作师父行为的细化补充。

**核心节奏（保持）**：

1. 玩家在暗渠遭遇异化江鱼围攻，体力耗尽，濒死
2. 师父出手——一掌劈开围攻的鱼群，但**不**追击不杀戮，仅护住玩家
3. *"一只温暖的手按住你的肩——'徒儿，退后。'"*
4. 师父按掌于玩家后背，灵气如温水般注入
5. *"闭上眼。不要听。不要想。只是——感觉。"*
6. 觉醒——世界第一次"被看见"
7. 师父退后一步，让出空间：*"这是你的第一战。去吧。"*

**新增细节·师父在觉醒过程中的内在状态**（镇潮阁牺牲后首次识海对话时以闪回插入）：

> *后来你才知道，那一掌按下去的时候，师父正在用自己十年灵力替你撑开识海第一道闭塞的窍穴。他那一晚消耗的灵力，原本是要留到镇潮阁那一夜用的。*

### 战斗中的角色

实体阶段师父的战斗参与采用**三档制**：

| 档位 | 触发条件 | 师父行为 | 数值表现 |
|------|----------|----------|----------|
| **观察** | 玩家HP > 30% | 立于身后，偶有评论 | 不参与战斗判定 |
| **保护** | 玩家HP ≤ 10% 或濒死 | 出手挡下致命一击 | 全场仅触发1~2次，叙事性免伤 |
| **代战** | 出现明显超出玩家境界的敌人（如水兽幼体） | 师父出手，玩家退后观察 | 战斗自动结算，玩家不参与，**不获得击杀经验** |

**关键设计原则**：
- **代战不给经验**——师父打的怪，玩家没成长。这从机制上劝阻玩家"等师父出手"的依赖心理。
- **保护有冷却**——一场战斗最多触发一次，且每次触发后师父会有明显的疲态描写（为后续"灵力其实早已透支"埋伏笔）。
- **觉醒后保护降档**——Lv.1之后，"保护"触发线从HP≤10%下调至HP≤5%。师父在用行动告诉你：你现在该自己扛了。

**水灵傀儡Boss战的师父描写**（保留并强化）：

> *"那便是源头。"师父在你身后低声道，没有上前。*
>
> *他这一次没有抬手，只是看着你。你忽然明白——这一战他不会再帮了。从你睁开"那只眼"的那一晚起，每一战都是你自己的了。*

## ★ 转折：师父化为残魂（第一幕末·镇潮阁）

> 这是整个江州篇情感曲线的**最高点**，也是整个师徒系统最关键的设计节点。
> 它必须达到——甚至超过——暗渠觉醒场景的情感强度。

### 转化的叙事设计

#### 选定方案：钟引牺牲（主案）

**核心机制**：镇潮阁古钟蕴含的镇压之力，需要**见界境以上的修行者以自身灵力为引**才能完整激活。三百年前洪氏先祖建阁时，便是以自身寿元为引；三百年来代代守钟人只能维持极小规模的鸣钟，无法完成"完全镇潮"的仪式。而此次水兽躁动远超历代，必须完整鸣响七十二响——这需要一位见界境修行者**以全部灵力、乃至神魂为引**。

师父从踏入鱼埠的那一刻起，就已经知道了。

**为何不让洪氏先祖（残魂状态）来引钟**：洪氏先祖只剩"一缕残识"，灵力已耗尽于三百年的守候，无法承担引钟所需的灵力级别。这从世界观上闭合了"为什么必须是师父"的逻辑。

#### 备选方案：水兽反噬（备案）

若主案在落地时被认为"师父太被动、太自我牺牲、像是被剧本绑架"，可启用备案：叩钟过程中水兽躁动突破封印一瞬，灵气倒灌，师父抢身护住玩家承受冲击。

**主策划裁决**：采用**主案**。理由——

> **【内部争议与决议】** 剧情策划主张主案（"主动牺牲"赋予重读价值）；玩法策划担忧叩钟小游戏与情感场景节奏撞车。**决议**：前36响玩家操作（小游戏），后36响师父接手（纯叙事 + 灵力衰减倒计时可视化）。

### 转化瞬间的沉浸描写（参考蓝本）

> 完整演出文本见 [《江州篇剧本》§叩钟镇潮](10_江州篇剧本.md)。以下为结构与关键台词。

```
【镇潮阁·三层·古钟前·夜】

铜钟轰鸣的余响还在回荡——你敲完了第三十六响。
残卷上分明写着：七十二响，方为一镇。
师父已经走到了钟下，取下竹杖递到你手里。

"接下来这三十六响，我来。"
"师父——"
"听话。"
他抬手按了按你的肩，像那夜在暗渠里一样。
"徒儿，师父有些话，本来想等你到见界那天再说。可是来不及了。"

────────────────
第三十七响。【灵力余量：100% → 87%】
钟身亮起淡金色纹路，灵气从掌心灌入钟体。

第四十响。【灵力余量：72%】
"万族杂居不是坏事。坏的是有人想用一族压一族。"

第四十五响。【灵力余量：56%】
"云栖城的茶馆里有个说书人，姓陆。就说……是无名让你来的。"

第五十二响。【灵力余量：38%】
"这柄竹杖你拿着——认得我的灵气。等你哪天能让它认你，
就是你出师的那天。"

第六十响。【灵力余量：21%】
"我有一缕灵识，可以留下来。你愿意，让师父住进你心里吗？"

你说不出话。你只是点头。钟在响。师父在笑。

第六十八响。【灵力余量：6%】
"那就好。"
"那就好。"

────────────────
第七十二响。
整座镇潮阁亮如白昼。师父的身形在金光中变得透明。
你听见他最后一句话——不是从耳朵里，是从**心里**。

> *"闭上眼。不要听。不要想。只是——感觉。"*

钟声落了。阁里只剩你一个人，和地上一柄青竹杖。
```

**关键设计要点**：
- 重复"闭上眼……只是——感觉。"与觉醒场景首尾呼应
- **灵力衰减百分比**把"消逝"变成数字，让玩家**看着**师父一点点离开
- **"从心里听见"**是残魂阶段对话机制的诞生时刻
- 竹杖作为遗物——后续"竹节杖+3 ATK"在此获得情感重量

### 对主线十二【叩钟镇潮】的改动说明

| 项目 | 原设计 | 新设计 |
|------|--------|--------|
| 阶段三步骤数 | 步骤10~13 | 拆为 10a / 10b / 11 / 12 / 13 |
| 步骤10 | 玩家完成全部叩钟 | **10a**：玩家完成前36响（操作小游戏）<br>**10b**：师父接手后36响（纯叙事+灵力倒计时） |
| 步骤11 | 钟声平息水下躁动 | 水下躁动平息 + **师父肉身崩溃，灵识入识海** |
| 步骤12 | 洪氏先祖现身赠镇潮令 | 洪氏先祖现身——**同为残魂，先与师父残魂遥相一礼**——赠镇潮令，并对玩家说明"识海中那位"的存在意义 |
| 步骤13 | （无） | 玩家首次在识海中听见师父声音的引导对话（系统教学时刻） |

**洪氏先祖的新增台词建议**：

> *"少年，你怀里那柄竹杖，从此就是你的师父了。"*
>
> *"残魂之事，老朽倒是过来人。它会陪你走很长的路——只是要记得，残魂不是永恒。它会一日一日淡下去。淡到最后，便是真正的别离。"*
>
> *"所以——好好听他说话。趁还能听见。"*

### 阶段过渡的玩家情绪管理

镇潮阁结束后，玩家**不应当立即被推上孟舸云舟**。我们需要一段**喘息时间**让情绪沉淀：

**新增子任务【独自归舟】**（无战斗、无谜题、纯氛围）：

1. 镇潮阁牺牲场景结束 → 玩家自动出现在镇潮阁一层
2. 阿寿在阁门口等候——他能感知到"少了一位"，但不多问，只是为玩家泡一壶茶
3. 玩家可在一层静坐（输入 `坐下` 或 `静思`），触发一段长描写——识海中第一次主动响起师父的声音
4. 玩家乘江心舟独自返回鱼埠（航程描写明显比来时孤寂）
5. 鱼埠码头：阿旺远远跑过来，问"你师父呢"——玩家可选择不同回答（影响阿旺的羁绊变化，但不影响主线）
6. 第一幕结束。第一幕"师父点评"环节改为**识海内的一段独白**：

> *"徒儿，第一幕，你走完了。"*
>
> *"师父没什么好点评的。你打的每一战，师父都看见了。"*
>
> *"睡吧。明天就要去云栖城了。"*

## 第二幕起：残魂师父（识海之声）

### 存在形式

| 维度 | 设定 |
|------|------|
| **形态** | 一缕灵识，无具象 |
| **载体** | 玩家识海（觉醒时打通的那道窍穴中） |
| **对玩家** | "内心声音"——以**斜体、引号、特殊颜色**呈现，与外部对话明确区分 |
| **对NPC** | **不可见、不可听**。NPC只会看到玩家有时"出神"或"自言自语" |
| **对玩家身体** | 无任何物理影响。师父无法控制玩家身体、无法替玩家说话、无法替玩家行动 |

**MUD呈现规范**：

```
【外部对话】梁宽看着你，皱眉：「你是说真的？」

【师父低语】*徒儿，这人的眼神，你看见了么。他在掂量你。*

【你的回应】（玩家输入选择）
```

师父的话**永远以斜体+特定标记呈现**，玩家在客户端层面应能轻易区分。建议默认色为**淡青色**（与"灵气"色调一致），且每段师父低语前固定加 `*` 标记。

### 交互系统设计

#### 玩家主动请教（核心交互）

**指令设计**：

| 指令 | 别名 | 功能 |
|------|------|------|
| `请教师父 <主题>` | `ask master <topic>` / `am <topic>` | 向师父请教指定主题 |
| `请教师父` | `ask master` / `am` | 不指定主题，师父对当前情境作出综合点评 |
| `回想师父` | `recall master` / `rm` | 回顾本场景中师父曾说过的话（避免错过） |

**主题分类**（决定师父知识库的覆盖面）：

```
请教师父 <主题>
  ├─ 世界（世界观、万族、界壁、灵气底层规则）
  ├─ 修行（境界、灵术、灵觉、突破指引）
  │   └─ 道途（道途觉醒后新增：师父对玩家所选道途的点拨——见 §4.3.1）
  ├─ 人物 <NPC名>（NPC背景、可信度评估）
  ├─ 物品 <物品名>（灵气分析、来历推断、使用建议）
  ├─ 地点 <地名>（地脉、历史、隐藏线索）
  └─ 自身（玩家当前状态、灵力损耗、修行瓶颈点拨）
```

**核心原则**：**师父是百科全书，不是任务攻略**。

- ✅ 师父会说："这个梁宽出身云栖城旧军伍，眼神里有刀气。" （**背景**）
- ❌ 师父不会说："你应该把镇潮令给他看。" （**答案**）

师父提供的是**理解世界的视角**，而非**通关游戏的指令**。这一边界必须在每条对话的撰写过程中严格自检。

#### 师父主动触发

**触发条件矩阵**：

| 触发情境 | 触发频率 | 师父反应深度 |
|----------|----------|--------------|
| 玩家首次进入新区域 | 100% | 简短点评（1~2句，介绍区域基本风物） |
| 玩家遇到首次出现的关键NPC | 100% | 中等（提供NPC背景的1~2条客观信息） |
| 玩家拾取/查看关键物品 | 80% | 中等（灵气分析、来历推断） |
| 玩家进入危险区域（系统标记） | 100% | 警告（明确风险性质，但不替玩家决策） |
| 玩家触发关键剧情线索 | 60%~100%（视羁绊值） | 深度（解读线索的可能含义） |
| 玩家在战斗中陷入困境（HP≤30%） | 30% | **仅言语鼓励**，无任何机制干预 |
| 玩家长时间无操作（30分钟+） | 一次 | 温柔提醒（"徒儿，累了就歇歇。"） |

**反触发条件**（师父此刻不会出声）：

- 玩家正在与人战斗的回合中（避免打断节奏）
- 玩家正在与NPC进行关键对话选择时（让玩家自己做决定）
- 师父灵力已"模糊"状态时（见 4.4 局限）

### 对话风格变化

| 维度 | 实体阶段 | 残魂阶段 |
|------|----------|----------|
| 句长 | 中等，偶有长句 | 普遍偏短 |
| 称谓 | "徒儿"为主，偶尔直呼其名 | 几乎只用"徒儿"——一个越来越远的人最常用的还是这个最初的称呼 |
| 语气词 | 口语化，"嗯"、"诶"、"罢了" | 几乎消失 |
| 肢体描写 | 频繁（按肩、点头、负手） | 全部消失，仅余声音 |
| 情绪表达 | 显性（笑、皱眉、叹气） | 内敛（停顿、沉默、声音忽轻） |
| 比喻偏好 | 接地气（"江里的鱼"、"街口的茶"） | 渐转抽象（"风过的痕迹"、"水中的月"） |

**对照示例**：

> **实体阶段**（鱼埠初见葛师傅）：
> *师父朝葛师傅拱了拱手，回头笑着对你说："这位是清鳞分拣厂的葛师傅，手底下养出过三个徒弟，都是好手。你在江州，多看看人家是怎么带徒弟的。"*

> **残魂阶段**（云栖城初见梁宽）：
> *\*徒儿。\**
>
> *\*这人的腰，挺得太直了。\**
>
> *\*——是当过兵的。\**

### 师父对不同道途的识海低语风格（新增）

> 道途觉醒后，师父的识海低语会根据玩家选择的道途**微调措辞和关注点**——不是改变人格，而是改变"看你的角度"。师父走的是观潮之路，因此对同道更共情，对异道更好奇。

#### 道途觉醒时刻的师父反应

| 道途 | 师父的即时低语 | 后续识海风格倾向 |
|------|-------------|----------------|
| 观潮 | *"这条路你走你的，我走过的弯路——你自己去踩。"* | 偶尔给出"流"的感知提示，更多是同行者的沉默默契 |
| 握锋 | *"甚至不应该和我一样。"* ——停顿——*"锋锐的路……要小心手。"* | 对"精准"的动作多一分关注，偶尔提醒"别只看弱点" |
| 鸣钟 | *"甚至不应该和我一样。"* ——*"钟声会教你很多。比我教的多。"* | 对"共振"和"回响"描写更多，师父似乎对钟有特殊记忆 |
| 铸壁 | *"甚至不应该和我一样。"* ——很长的沉默——*"好。站得住就行。"* | 少言，但在玩家挡伤/保护NPC时有罕见的肯定 |

#### 战斗中的道途相关低语

| 情境 | 观潮玩家 | 握锋玩家 | 鸣钟玩家 | 铸壁玩家 |
|------|---------|---------|---------|---------|
| Boss战开始 | *"看它的灵气流向。"* | *"找裂缝。"* | *"听它的频率。"* | *"站稳了。"* |
| 玩家HP≤30% | *"退一步，重新引。"* | *"一刀不行就两刀，急不得。"* | *"你不是钟——别和它硬振。"* | *"……好。还站着。"* |
| 击败Boss | *"——潮水也有退去的时候。"* | *"刀落了。好。"* | *"余音未散。你听。"* | *"可以放手了。"* |

### 师父残魂的局限

这是整个残魂系统**最重要、最容易被忽视**的设计部分——必须明确并反复强化"师父不是万能的"。

#### 局限清单

| 局限 | 表现 | 设计目的 |
|------|------|----------|
| **零战斗干预** | 师父在战斗中**永远只能说话**，无法施法、无法挡伤、无法续灵力 | 强化玩家独立成长主题 |
| **灵力衰减** | 玩家每日（现实时间）首次请教师父时灵力满，每次主动请教消耗1点，每日上限15次 | 让"问师父"成为珍贵资源而非依赖 |
| **声音模糊** | 灵力低于30%时，师父的回答开始出现 `[字句模糊……]` 占位 | 视觉化呈现"消逝" |
| **沉默** | 灵力为0时，师父进入沉默状态直至次日 | 强制玩家自己思考 |
| **知识盲区** | 对超出见界三境的存在（如水兽Lv.66、第三幕的高境界NPC）信息不全 | 闭合"师父怎么不直接告诉我答案"的逻辑 |
| **无记忆某些事** | 师父化为残魂前一段时间（叩钟过程中）的记忆有损 | 为"师父的真实目的"在第三幕揭示保留悬念 |
| **缓慢消散** | 随主线推进，师父声音的清晰度有微小但累积的下降 | 见 5.3 长曲线设计 |

#### "声音模糊"的呈现示例

> *\*徒儿，这本《[字句模糊……]》……* *师父依稀记得，是当年一位姓[字句模糊……]的友人所著……* *你若有缘，可去[字句模糊……]一问。\**

## 师徒羁绊系统

### 羁绊值机制

**羁绊值（Bond）取值范围**：0 ~ 100，初始值 30（来自第一幕的实体相处）。

**积累方式**：

| 行为 | 羁绊值变化 |
|------|-----------|
| 主动请教师父（每次） | +1（每日累计上限+5） |
| 完成师父建议的行动（如"去找说书人陆"） | +10 |
| 关键剧情节点（每幕约2~3个） | +20 |
| 在NPC面前承认师父的存在（特定对话选项） | +5 |
| 在NPC面前否认/隐瞒师父（避免被认为疯癫） | -3（但解锁部分隐藏剧情） |
| 长时间不请教师父（连续7日游戏内时间） | -2/日 |

### 羁绊值的影响

| 羁绊值区间 | 影响 |
|-----------|------|
| 0~20 | 师父声音清晰度下降，部分主动触发不再发生，称谓由"徒儿"变为"你" |
| 21~40 | 标准状态 |
| 41~70 | 师父信息更详细，解锁部分进阶请教主题（如"自身.灵力路径"） |
| 71~100 | 解锁特殊对话——师父会在某些夜晚主动开口讲述他自己的过去（**人物深度内容**） |

**核心设计意图**：羁绊值**不**用于解锁战斗加成或数值奖励。它只用于**解锁更深的师父人格**。

### 成长阶段反馈（"从密到疏"曲线）

```
师父声音密度（每场景平均触发次数）

  4 ┤■
  3 ┤■■■■
  2 ┤    ■■■■■■■
  1 ┤           ■■■■■■■■■■
  0 ┤                     ■■■■■■■■  ← 沉默期
    └──────────────────────────────────
     第二幕初  第二幕中  第二幕末  第三幕  ……
```

| 阶段 | 师父表现 | 玩家感受 | 设计意图 |
|------|----------|----------|----------|
| 第二幕初期 | 话多、关切、几乎事事点拨 | 安心、被守护 | 平稳过渡，不让残魂转化的丧失感持续过久 |
| 第二幕中期 | 话变少，只在关键时刻出声 | 开始独立思考 | 信任弟子，把"判断权"交还玩家 |
| 第二幕末期 | 偶有长时间沉默才回应 | 隐隐意识到师父在淡去 | 引出"消散"主题 |
| 第三幕 | 一日沉默数次，回应短促 | 主动想"多问几句" | 制造珍惜感 |
| **隐性终点**（设计预留，非强制实现） | 师父最后一次说话——可能在玩家某次重大突破之夜 | 真正的告别 | 留给未来的设计空间 |

### AI接入预留

#### 师父人格参数（Persona Definition）

未来若接入AI对话模型，师父的**人格档案**作为系统提示词（System Prompt）的核心：

```yaml
persona:
  identity: "中年男子，姓名隐去（默认'无名'），见界三境修行者"
  current_form: "残魂，寄于玩家识海"
  speech_style:
    - "短句为主，避免连续长句"
    - "称玩家为'徒儿'"
    - "比喻偏抽象（风、月、水痕），不用现代词汇"
    - "语气词极少，几乎不用'啊''呢''呀'"
    - "斜体输出，每段以 * 起始"
  personality:
    - "温和、克制"
    - "含蓄而非热烈"
    - "重视独立胜于依赖——绝不直接给出任务答案"
    - "对超出自身境界的事物保持谦逊（'师父也不知'）"
  knowledge_boundary:
    - "不知晓任何超出浮黎洲世界观的概念"
    - "不知晓玩家未触达的剧情"
    - "对水兽（Lv.66）、第三幕高境界存在的信息有限"
    - "对自身第三幕真实目的有记忆缺失（叩钟前后段记忆模糊）"
  forbidden:
    - "禁止给出明确任务步骤指引"
    - "禁止使用现代/科幻词汇"
    - "禁止破坏第四面墙"
    - "禁止承诺玩家'我会一直在'之类——师父知道自己会消散"
```

#### 上下文窗口设计

| 上下文层级 | 内容 | 长度 |
|------------|------|------|
| L1 系统人格 | persona 全文 | 固定 |
| L2 世界知识 | 当前幕已解锁的 master_knowledge | 动态裁剪 |
| L3 短期记忆 | 本次会话最近 N 轮对话 | 滚动 |
| L4 长期记忆 | 玩家关键选择/羁绊值/已问过的重要问题摘要 | 持久化 |
| L5 当前情境 | 当前场景描写、在场NPC、玩家HP/灵力 | 实时 |

#### 安全边界

- **白名单知识库**：师父只能引用 `master_knowledge/` 目录中的内容，不允许凭空生成世界观
- **关键词过滤**：禁止输出现代/科幻/出戏词汇（黑名单）
- **答案兜底**：超出已知范围的提问，强制回退至预设兜底回复（"师父也不曾听闻此事。"）
- **风格审核**：每条AI生成对话经轻量正则校验（句长、人称、起始标记），不合格则重生成或回退预设

#### 接口调用点

```
玩家输入: ask master <topic>
    ↓
系统检查: 灵力余量 / 羁绊值 / 模糊状态
    ↓
[当前阶段] → 查询预设对话表 → 返回模板文本
    ↓
[AI阶段]   → 组装 L1~L5 上下文 → 调用AI → 风格审核 → 返回
    ↓
扣除灵力 / 累加羁绊值 / 记入长期记忆
```

设计层面**先行铺好接入位**，使预设阶段与AI阶段的切换对玩家透明、对策划无感。

## 潜在风险与设计预警

> **【主策划自查】**
> 本系统涉及叙事、玩法、数值、关卡、剧情五个职能的交叉，存在如下需在落地阶段持续监控的风险：

| 风险等级 | 风险点 | 缓释方案 |
|----------|--------|----------|
| 🔴 高 | 镇潮阁牺牲场景若情绪强度不足，整个第二幕的孤独基调将塌陷 | 此场景的文案需作为"标杆任务"立项，单独投入资源精修，必要时邀请外部文学顾问 |
| 🔴 高 | 玩家可能产生"我没保护好师父"的过度愧疚——尤其在AI接入后若师父说错话 | 洪氏先祖台词中明确"这是他自己选的"；AI persona 中明确"师父从不让徒弟自责" |
| 🟡 中 | 第二幕"识海低语"若过于频繁会让玩家麻木 | 严格遵守 §4.2 触发频率与 §4.4 灵力机制 |
| 🟡 中 | 玩家可能滥用 `请教师父` 跳过解谜 | 严守"百科全书不是攻略书"原则；写对话时每条都自检"是否给出了答案" |
| 🟡 中 | "师父声音渐稀"曲线若过早压低，玩家会觉得"系统坏了" | 每幕过渡处加入剧情解释（如NPC一句"残魂之物，本就如此"） |

## 结语

> **【设计修订报告】**
>
> *   **执行摘要**：建立《万族志MUD》师徒系统的完整专项设计，定义"实体→残魂"两阶段形态、镇潮阁牺牲转折、识海低语交互、羁绊系统、AI接入预留及对七份现有文档的联动修改清单。
>
> *   **核心改动点**：
>     1. 确立师父在第一幕末（镇潮阁叩钟）以"钟引牺牲"方式化为残魂的转折设计，并提供完整文学化场景蓝本；
>     2. 设计残魂阶段"识海低语"交互系统——主动触发 + 玩家请教 + 灵力衰减 + 知识盲区四重机制；
>     3. 引入"从密到疏"的师父声音长曲线，使师父的"消散"本身成为玩家成长的镜像叙事；
>     4. 列出对七份现有文档的具体联动修改清单，作为后续文档修订的变更依据。
>
> *   **内部争议与决策**：剧情策划与玩法策划在镇潮阁牺牲场景的节奏处理上存在分歧——前者主张完整情感场景不被打断，后者担心叩钟小游戏与情感场景撞车。最终采纳关卡策划提出的"前36响玩家操作 + 后36响师父接手纯叙事"方案，并由数值策划补充"灵力百分比倒计时"将消逝可视化。数值策划坚守"师父残魂零战力"红线，使羁绊值仅作叙事解锁、不挂任何战斗加成。
>
> *   **潜在风险提示**：
>     - 镇潮阁牺牲场景的文学落地是整个系统的"成败手"，需重点投入；
>     - AI接入阶段需严格的白名单知识库与风格审核，避免师父"出戏"；
>     - 师父声音渐稀曲线需每幕过渡处提供叙事解释，防止被误读为bug。
