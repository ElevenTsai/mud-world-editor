# 《万族志MUD》NPC设定总览

*——每一个NPC都有自己在乎的事*

---

## 设定总则

1. **NPC不是任务机器**：每个NPC都有独立的动机、日常作息和情感，玩家可以与之闲聊而非只触发任务。
2. **对话反映身份**：渔民说渔谚，商贾算账目，修士引经据典。年龄、阅历、性格都应体现在遣词用句中。
3. **时间敏感**：部分NPC在不同时段出现在不同地点，或展现不同状态（如陈老鱼头清晨在码头，午后在竹棚下）。
4. **关系网络**：NPC之间存在师徒、亲属、同行、交易等关系，玩家可通过一个NPC得知另一个NPC的信息。
5. **隐藏深度**：多数NPC拥有不会主动提及的秘密或过往，需要特定条件（好感度、物品、多轮对话）才会透露。

# 一、江州鱼埠

## 渔户营

### 陈老鱼头

| 属性 | 描述 |
|------|------|
| 所在 | 渔户营 |
| 年龄 | 七十余岁 |
| 身份 | 渔户营最年长的老渔民，非官方"村长" |
| 境界 | 闻道二境（Lv.2）——七十年江上生涯，灵觉已开，未曾正式修行 |

**外貌**：满脸风霜纵横如老树皮，花白头发用一根草绳胡乱束着。嗓门洪亮，说话时习惯用手拍大腿。穿着破旧但干净的粗布短衫，脚上永远是一双草鞋。

**性格**：热心、健谈、护短。对渔户营的年轻人视如己出，对外来人则先审视三分，觉得可靠后才会掏心窝子。骨子里是个念旧的人——对旧物、旧人、旧规矩都看得极重。

**背景**：在江州鱼埠生活了一辈子。十三岁随父出江，十八岁独驾渔船，经历过三次百年大潮。妻子在一次洪水中失踪，此后再未续弦，将整个渔户营视为自己的家。年轻时曾划船到镇潮阁附近，听见过水底传来的钟鸣，自此再不敢靠近。这段经历他从不主动提起，但若玩家带着镇潮阁相关物品回来问他，他会沉默良久后缓缓道来。

**日常行为**：清晨在码头边检查渔网绳结，午后坐在竹棚阴凉处给晚辈讲掌故，傍晚独自站在栈道尽头望江，嘴里嚼着干鱼丝。

**对话风格**：爱用渔谚——"三月鱼肥四月虾，五月螃蟹往上爬"。说到兴起时拍大腿，说到伤心处叹一口长气。语速不快，但条理分明。

**游戏功能**：主线引导NPC。提供江州鱼埠背景信息与风土人情，指引玩家了解各区域，触发【鱼埠初至】主线。

**关键关系**：
- 与赵船主是四十年的老邻居，看不惯赵船主爱吹牛但暗地里关心他。
- 与洪大爷年轻时一起出过海，知道洪家与镇潮阁的渊源但守口如瓶。
- 视阿鲤如孙女，常叮嘱她别跑太远。

**性格**：活泼、好奇、话多。对外来人充满好奇，但被陌生人盯着看时会突然害羞，假装去整理鱼篓。有一点小小的虚荣心——喜欢听人夸她鱼选得好。

**背景**：父亲是渔户营普通渔民，母亲病弱在家。阿鲤每天天不亮就帮父亲把鱼挑到鱼市卖，回来后还要照顾母亲。她从没离开过鱼埠，对远方的世界充满向往，常站在栈道上望江发呆。

**游戏功能**：气氛NPC，提供鱼埠生活的鲜活感。可触发小型互动（如帮她选鱼、听她讲鱼埠趣事）。

| 属性 | 描述 |
|------|------|

| 境界 | 闻道一境（Lv.1）——多年出江，偶有灵觉，本人浑然不知 |

**外貌**：微胖，面色红润，穿着比普通渔民讲究几分的绸面褂子。手指粗短但保养得干净，说话时爱晃动腰间的铜钥匙串。

**性格**：爱吹嘘、好面子、但本性不坏。嘴上说见过水妖，其实半真半假——他确实在江上见过一次不明黑影，此后越传越邪乎，自己也信了三分。当手下船员失踪时，他的焦急和自责是真实的。

**背景**：祖上三代都是渔民，靠攒钱逐渐置办了三条船，在鱼埠算得上殷实。一直想把生意做到云栖城去，但江道不宁让他寸步难行。

**日常行为**：上午在码头指挥卸鱼，下午去栖浪酒家喝两杯吹嘘"当年勇"，晚上在家数账。

**对话风格**：嗓门大，爱拍胸脯保证——"这事包在我赵某人身上"。吹牛时眉飞色舞，被揭穿时假咳掩饰。

**游戏功能**：触发【失踪的渔船】支线入口。任务完成后提供感谢奖励，并成为后续商路支线的潜在NPC。

## 江州鱼市

### 郑掌柜（郑记鱼行）

| 属性 | 描述 |
|------|------|

| 境界 | 闻道一境（Lv.1）——常年经手灵鱼，指尖可辨灵气成色 |

**外貌**：精瘦干练，手里永远捏着一把磨得发亮的算盘，眼神极锐利——号称"一眼辨三两"。穿蓝布围裙，袖口挽得老高。

**性格**：精明、公道、嘴快。对生意锱铢必较但不会骗人，坊间信誉极好。说话直来直去，不留面子。

**背景**：少年时跟着父亲在鱼市跑腿，摸爬滚打二十余年，如今是鱼市最大的鱼行老板。对江州每种鱼的价格行情烂熟于胸。

**对话风格**：说话噼里啪啦像打算盘——快、准、不废话。

**游戏功能**：信息NPC，提供江中异象的鱼市视角情报。可做为买卖交易NPC（鱼类交易）。

| 属性 | 描述 |
|------|------|

**外貌**：身形瘦削但筋骨极硬，双手布满老茧和细密的鱼鳞刀疤。一把刮鳞刀用了二十年，刀刃薄如纸。面无表情时像一截枯木。

**性格**：沉默寡言、手艺至上。说话不超过十个字，但手上的活儿从不含糊。对徒弟阿旺嘴上严厉，实则处处照看——每次阿旺被鱼刺扎了手，是他默默把药膏放在旁边。

**背景**：十五岁入行，在清鳞分拣厂干了整整四十年。他的师父就是鲁伯的师父所造刮鳞刀的使用者，两家手艺有渊源。妻子嫌他身上鱼腥味走了，此后孑然一身，把所有心思都放在手艺上。

**对话风格**：能用三个字说完的绝不用四个——"嗯""去""不行""可以"。偶尔被问到手艺问题时，会难得地多说两句。

**游戏功能**：氛围NPC，体现"手艺人"的匠人精神。侧面烘托师徒主题。

## 捕鱼房

### 孙老六

| 属性 | 描述 |
|------|------|
| 所在 | 捕鱼房 |
| 年龄 | 五十来岁 |
| 身份 | 退休渔民，现在补网为生 |
| 境界 | 闻道二境（Lv.2）——年轻时溺水曾与水兽灵气短暂接触，灵觉由此开启 |

**外貌**：身材高大但左腿微跛，走路一高一低。面容沧桑，手指粗壮灵活——补起渔网来飞针走线。嘴里总哼着曲调低沉的渔歌。

**性格**：豁达、怀旧、讳莫如深。年轻时遭过大难，但并不怨天尤人。爱哼渔歌，歌词里藏着祖辈传下来的秘密，但若被追问细节，就会笑着岔开话题。只有在特定条件下才会正色相告。

**背景**：年轻时是鱼埠最矫健的渔夫，一次深水捕鱼被大鱼拖下水，伤了筋骨落了跛。那次溺水的瞬间他看见了水底的黑暗中有巨大的影子缓缓游动。他把这事藏在渔歌里——"镇潮阁下水深深，锁着老龙不翻身"——用歌谣的方式一代代传递，既不会被遗忘，也不会引起恐慌。

**日常行为**：白天在捕鱼房补网唱歌，傍晚坐在门槛上看江面落日。

**对话风格**：悠悠然的慢性子，说话爱用比喻——"这网就跟人心一样，断了得赶紧补"。

**游戏功能**：提供【渔歌里的故事】主线线索。赠予特制的尾橹（关键任务道具）。是连接民间传说与镇潮阁真相的桥梁人物。

| 境界 | 闻道三境（Lv.3）——所学不全，仅存符箓皮毛 |

**外貌**：满脸皱纹，缩在摊位后的小马扎上。手指骨节粗大但画符时稳如磐石。面前摆着一沓黄纸符，用石头压着。

**性格**：沉默、神秘、看似迷糊实则清醒。旁人都当她是骗钱的江湖把戏，只有素娥看出那些符上确有微弱灵气。

**背景**：年轻时据说是某位散修的弟子，学了些皮毛符箓之术。修为几乎为零，但画符的手法中保留着正统的灵气引导路径。她自己或许都不完全明白原理，只是按照师父教的一笔一划地画。这些符确实有微弱的护体效果——尤其在面对灵气冲击时。

**对话风格**：含糊低语——"买一张吧……保平安的……"被细问时只笑不语。

**游戏功能**：支线商品NPC。护身符在通玄门面对水兽威压时自动触发一次护体效果（实用道具）。

## 通玄门

### 守门修士陆清玄

| 属性 | 描述 |
|------|------|
| 所在 | 通玄门 |
| 年龄 | 三十余岁 |
| 身份 | 筑基期修士，受宗门派遣驻守通玄门 |
| 境界 | 通幽三境（Lv.22）——江州篇境界最高的修士 |

**外貌**：身着玄色道袍，面容清俊但带着淡漠。腰佩一柄无鞘短剑，周身隐隐有灵气波动。站在通玄门前如同门上的雕塑般静默。

**性格**：淡漠、尽职、内有慈悲。表面上对一切事务都以"职责"二字衡量，实则骨子里有一颗柔软的心——否则不会在听到水兽的遭遇后被说服选择化解而非镇压。作为城中修为最高者，他习惯了独自承担重压。

**背景**：出身中等修行宗门，因某些不为人知的原因被派到这座偏远的通玄门驻守——在宗门中不算重用，更像是一种流放。但他并不抱怨，反而在这里找到了一种安宁。筑基期的修为在修行界不算高，但在云栖城已是最强者。他对界壁、灵脉、古阵法有深入的学术性了解。

**日常行为**：清晨在广场打坐修行，白天维护传送法阵，夜间巡视阵纹。近日频繁感知崖底灵气异动，精神高度紧绷。

**对话风格**：简洁而沉——"说你们发现了什么。""这不是普通的水兽。三百年了。"做出最终决策时语气坚定——"若你所言属实……我愿一试。"

**游戏功能**：主线终幕核心NPC。提出镇压与化解两种方案，最终被说服选择化解。赠予"通玄令"（前往下一区域凭证）。是江州篇的关键决策引导者。

### （隐藏）水兽之灵

| 属性 | 描述 |
|------|------|
| 所在 | 通玄门广场边缘（条件触发） |
| 年龄 | 上古存在 |
| 身份 | 江底被镇压的远古水兽灵体投影 |
| 境界 | 铸界七境（Lv.66）——上古灵兽，远超当世修行者 |

**外貌**：巨大的虚影，形似长蛇与鲸鱼的结合体，通体深青色鳞光隐现。面部模糊但能感受到悲怆而非凶恶的气息。显现时周围灵气剧烈波动，空气变得潮湿沉重。

**性格**：悲怆、克制、守护者本能。三百年的囚禁并未让它变得暴戾——它的躁动只是因为它所守护的灵脉日渐枯竭，而封印让它无能为力。它不恨镇压它的人，只恨自己无法完成职责。

**背景**：上古时期即守护江底灵脉的灵兽，与浮黎洲同龄。三百年前被误认为兴风作浪而遭封印——实际上当年的风浪正是它在疏通灵脉时的副作用。封印后灵脉无人照料，逐渐淤塞衰竭。每隔百年封印松动时它会尝试挣脱以恢复职责，但每次都被当作"水妖躁动"而重新镇压。

**对话风格**：低沉、回响，像从深水中传来——"吾……非为害世间……灵脉……将枯……"

**游戏功能**：江州篇最终"BOSS"与核心叙事转折。化解路线中通过对话揭示三百年误会的真相，赠予"水灵珠"（饰品，水性亲和）。临别提示灵脉向北延伸至江州腹地更深处（伏笔）。

**触发条件**：集齐《镇潮录》残卷、深青色鳞片、《云栖幽影图》三件关键道具，了解事件全貌后，在通玄门广场边缘出现。
