# 《万族志MUD》战斗数值设计

*——剑出鞘前，先观潮汐；招出手时，已落十分。*

---

## 系统概述

### 设计哲学

> **"战非好战，战为不得已。每一场战斗，都是一段叙事的呼吸。"**

《万族志MUD》的战斗系统不是数字对撞的擂台，而是叙事节奏中的**重音节**。玩家在江雾、潮声、灵气流转之间举刀，每一次出手都应让世界更**清晰**一分，而非更**麻木**一分。

- **战斗服务于叙事**：异化江鱼并非EXP包，而是"江水生病了"的具象证据；齿轮傀儡的鳞片不是数值墙，而是江州沦陷的工业噪音。
- **指令制（MUD传统）**：玩家以 `attack`/`cast`/`defend`/`flee`/`item` 等动词与战斗交互，每一条指令背后皆有可被文字描写的余韵。
- **伪回合制**：行动顺序由 **速度（SPD）** 决定，而非严格"我方→敌方"。同等速度时，**主动方先手**（玩家若主动 `attack`，先行）。
- **文字为体，数值为骨**：每一次伤害结算同时输出 ①数值结果（HP变化）②文字描写（动作与感官）。无文字描写的战斗不予通过。

### 设计原则

1. **公式安全**：所有除法采用 `A * K / (K + B)`，K=50。结构性杜绝除零、杜绝指数膨胀。
2. **线性 / 微对数增长**：所有新设计属性（攻、防、灵攻、灵抗、速度）随等级**线性递增**。Lv.10 玩家面对 Lv.5 敌人有压制感但非"一击碾压"。
3. **回合数底线**：普通杂兵战 **3~5 回合**；小Boss **5~7 回合**；中Boss **6~9 回合**；大Boss **8~12 回合**。低于 2 回合视为"无意义战斗"，应改为叙事描写而非战斗实现。
4. **资源与策略并重**：单纯普攻可胜，但消耗资源；用对灵术可显著缩短回合；战斗收益按"使用资源-换取效率"线性结算。
5. **境界压制有上限**：等级差→伤害修正封顶 ±50%/-30%，避免"Lv.66 vs Lv.1 = ∞" 的数学怪兽，由叙事承担"绝对差距"。

## 基础属性体系

### 属性总览

> 设计目标：**HP、灵力沿用《玩家能力设计》既有数据**；新增 **攻击力（ATK）、防御力（DEF）、灵攻（SpA）、灵抗（SpR）、速度（SPD）** 五项，全部线性递增。

| 缩写 | 中文名 | 类别 | 作用 | 增长方式 |
|------|--------|------|------|---------|
| HP   | 生命值 | 沿用 | 归零即败 | 见《玩家能力设计》"HP表" |
| MP   | 灵力上限 | 沿用 | 灵术燃料 | 20 + 等级 × 15（标注以文档为准） |
| ATK  | 攻击力 | **新设** | 物理伤害基底 | 8 + 等级 × 2 |
| DEF  | 防御力 | **新设** | 减免物理伤害 | 5 + 等级 × 2 |
| SpA  | 灵攻 | **新设** | 灵术伤害加成（百分比） | 0 + 等级 × 3 |
| SpR  | 灵抗 | **新设** | 减免灵术伤害 | 0 + 等级 × 2 |
| SPD  | 速度 | **新设** | 行动顺序判定 | 10 + 等级 × 1 |

### 装备与外饰修正

闻道境无重型装备，外物影响轻微：

| 物品类型 | 修正范围 | 例 |
|---------|---------|---|
| 武器（柴刀/短桨/法杖） | ATK ±2 ~ +4 | 师父遗物"竹节杖"+3 ATK（镇潮阁后获得，叙事重量极高——详见 [《师徒系统设计》§3.2](09_师徒系统设计.md)） |
| 护身（粗布/皮甲/丝绦） | DEF ±1 ~ +3 | 洪氏家传丝绦 +2 DEF |
| 灵物饰品（玉佩/铜钱串） | SpA 或 SpR +2 ~ +5 | 素娥所赠"清音佩"+5 SpA |

## 伤害计算系统

### 公式总表

#### 物理伤害

```
基础物伤 = ATK_攻方
减免后    = 基础物伤 * 50 / (50 + DEF_守方)
浮动后    = 减免后 * (0.9 ~ 1.1)         ← ±10% 随机
境界修正  = 浮动后 * (1 + 等级差修正)     ← 见"境界压制"章节
最终伤害  = max(1, 境界修正)              ← 至少1点，避免"无效命中"
```

#### 灵术伤害

```
基础灵伤 = 灵术声明伤害（如灵气弹 8~12，取随机）
灵攻加成 = 基础灵伤 * (1 + SpA / 100)
减免后    = 灵攻加成 * 50 / (50 + SpR_守方)
境界修正  = 减免后 * (1 + 等级差修正)
元素修正  = 境界修正 * 元素克制系数        ← 见"元素克制"
最终伤害  = max(1, 元素修正)
```

> **数值策划注**：灵术不再叠加 ±10% 的物理浮动——灵术本身的"声明伤害"已经是范围（如 8~12），随机性已内置，避免双重随机导致方差过大。

### 暴击系统（简化）

- **基础暴击率**：5%（玩家），3%（普通敌人），8%（部分Boss）
- **暴击倍率**：×1.5
- **暴击触发**：仅普攻可暴击；灵术不可暴击（避免AOE暴击雪崩）
- **特殊触发**：`聚气一击` 充能后必定暴击（即倍率 ×1.5 已内含于"+12 伤害"的设计中，不再额外暴击）

> **设计意图**：MUD 的爽点不在数字跳动，而在文字"破甲"那一笔的描写。暴击率刻意压低，让暴击成为**叙事高光**而非例行公事。

### 元素克制（江州篇）

江州为水乡，元素体系仅启用 **水 / 火 / 雷** 三系。无元素的灵术（如灵气弹）视为**中性**，无加减。

| 攻方\守方 | 水 | 火 | 雷 | 中性 |
|----------|----|----|----|------|
| 水       | ×0.7 | ×1.0 | ×1.0 | ×1.0 |
| 火       | **×1.5** | ×0.7 | ×1.0 | ×1.0 |
| 雷       | **×1.5** | ×1.0 | ×0.7 | ×1.0 |
| 中性     | ×1.0 | ×1.0 | ×1.0 | ×1.0 |

> **关卡策划注**：水灵傀儡、触须幻影、水兽之灵均为水系——这意味着**火/雷系外援**（如陆清玄的雷符、师父的炉火灵符）在剧情关键战中拥有显著叙事意义，而非仅作"NPC助攻"。

### 境界压制修正表

| 等级差（攻方-守方） | 攻方伤害修正 | 守方伤害修正 |
|------------------|------------|------------|
| +5 及以上 | **+50%（封顶）** | -30%（封顶） |
| +4 | +25% | -18% |
| +3 | +18% | -12% |
| +2 | +12% | -8% |
| +1 | +6% | -4% |
| 0 | 0% | 0% |
| -1 | -5% | +4% |
| -2 | -10% | +8% |
| -3 | -15% | +12% |
| -4 | -22% | +18% |
| -5 及以下 | **-30%（封顶）** | +30%（封顶） |

> **主策划注**：等级差封顶 ±50%/-30% 是**世界观的妥协线**。剧情上"师父Lv.12不可能击败水兽Lv.66"由"不可战胜"标记承担（见后文），而非由数学无穷大兜底。

## 战斗AI行为模式

> 设计目标：让每类敌人在文字层面也具有**可识别的"性格"**，而非数值差异的傀儡。

### 1. 低等灵兽型（异化江鱼、寄生灵兽）

- **基本策略**：每回合 80% 普攻、20% 扑咬/啃噬（小幅额外伤害）。
- **目标选择**：HP 最低的玩家方单位；若仅一名玩家，则随机部位（用于丰富文字描写）。
- **逃跑机制**：HP < 20% 且非任务必杀目标时，30% 概率脱战逃逸（异化江鱼游回浊水中，叙事上呼应"潮水尚未退尽"）。
- **特征行为**：动作描写偏"野性、不规则"——咬、扑、缠。

### 2. 傀儡型（水纹石像、齿轮傀儡）

- **基本策略**：行为高度**程序化**，玩家可通过观察预判。
  - 水纹石像：固定循环 `[蓄力 → 重击 → 喘息]`，蓄力回合可被聚气一击中断。
  - 齿轮傀儡：固定循环 `[轻 → 轻 → 重]`；"重"前一回合发出齿轮加速声，给玩家预警窗。
- **目标选择**：固定锁敌（首位攻击者），不切换。
- **逃跑机制**：无（傀儡至损毁）。
- **特征行为**：描写偏"机械、可预测、冰冷"——"齿轮咬合""石屑剥落"。

### 3. Boss型（水灵傀儡、触须幻影）

- **阶段化（HP阈值触发）**：详见"Boss战设计规范"。
- **威胁判定**：每回合根据玩家当前威胁（已造成总伤+灵术使用频率）选择**最优反制**——
  - 若玩家持续灵术：使用降灵抗debuff或物理快攻。
  - 若玩家挂盾：使用穿透/范围。
- **目标选择**：动态切换至"最危险"目标。
- **特征行为**：描写偏"巨力、压迫、未知"——"水柱涌起""触须从云间垂落"。

### 4. 群体型（灵气异化虫群）

- **基本策略**：成员分散站位，单体威胁极低；以**总数与持续补充**形成压力。
- **目标选择**：随机分散（每只虫独立锁敌），保护NPC时虫群优先攻击 NPC。
- **逃跑机制**：环境净化（如灵气涟漪范围清场）后剩余虫群有 50% 概率溃散。
- **特征行为**：描写偏"密集、嗡鸣、令人不适"——"如灰色潮水""群翅震耳"。

## 战斗文字描写规范

> **剧情策划准则**：每一条战斗输出 = `[行动描写] + [感官细节] + [数值结果]`。三者缺一不可，否则视为占位符待补。

### 普攻模板

#### 命中（普通）
```
你挥竹杖，斜劈而下——杖梢挂着江雾，落处溅起一线水光。
> 异化江鱼受到 8 点伤害（HP 25 → 17）
```

#### 命中（暴击）

## 境界压制与平衡机制

### 等级差→实战影响表

| 等级差（玩家相对敌人） | 战术表现 | 文字基调 |
|--------------------|---------|---------|
| +5 及以上 | 玩家普攻 ≈ 1击秒；敌方伤害几乎可忽略；战斗 1~2 回合解决 | "你抬手，对方已散" |
| +3 ~ +4 | 玩家压制感明显，约 2~3 回合 | "你出手如风，对方应对吃力" |
| +1 ~ +2 | 略有优势，约 3~4 回合 | "你略占上风，但仍需谨慎" |
| 0 | 势均力敌，3~5 回合 | "你与之短兵相接，每一击皆需算计" |
| -1 ~ -2 | 玩家吃力，需用资源换胜，5~8 回合 | "对方每一击都让你后退半步" |
| -3 ~ -4 | 极险，需精准操作或外援，可能失败 | "你的灵气几近枯竭，仍未撼动对方分毫" |
| -5 及以下 | **不建议正面**：算上 ±50% 封顶，伤害仍会卡在底线，但回合数膨胀至难以承受 | "你抬头——这场战斗的尽头，不在你手中" |

### "不可战胜"标记

> **主策划准则**：当数学差距已无意义（如玩家 Lv.1~7 vs 水兽之灵 Lv.66），系统应**拒绝进入战斗**，而非强行计算。

#### 触发规则

```
若 (敌人等级 - 玩家等级) ≥ 10，则该敌人对玩家标记为【不可战胜】。
```

#### 系统响应

- `attack 水兽` → 输出叙事文本（而非进入战斗）：
  ```
  你抬起手——
  指尖刚触到那道笼罩你的庞大灵压，便如指尖触雪，瞬间湿冷至骨。
  你的灵脉根本无法承载这层"对话"，更遑论"对抗"。
  
  > 此目标的境界远超于你。你能做的，唯有"听"，或"问"。
  ```
- 该状态可随玩家境界递进而解除，但 ≥10 级差永远禁用 attack。

### 师父 Lv.12 vs 水兽 Lv.66 的叙事解释

> **剧情策划论证**：差距 54 级，远超 ±50% 封顶。师父对水兽的"对抗"在文本上呈现为**牵制 / 拖延 / 牺牲**，而非"战斗"。此数值差距正是**师父在镇潮阁燃尽灵力牺牲的数值前提**——他无法战胜水兽，只能以另一种方式（叩钟仪式）完成镇潮。

剧情中所有"师父挡下水兽"的桥段，**严禁触发战斗系统**，应以**演出文本 + 时间限制小游戏**呈现：

> 师父反握竹节杖，将左手三指按在杖尖——
> 一道极细的灵脉自他指尖延出，缠住水兽垂落的触须。
> "我撑不了多久。"他声音平稳，眼底却已有血丝渗出。
> "去吧——洪老所言的灵脉源头，就在你左前方的暗道里。"
> 
> [限时事件] 你有 3 回合时间穿过暗道，否则师父将重伤撤退。

### 越级挑战：风险与奖励

| 等级差 | 经验加成 | 掉落加成 | 风险 |
|-------|---------|---------|------|
| -1（敌方高1级） | ×1.5（结合既有"高1级敌×18 EXP"公式） | 普通 | 中等 |
| -2 | ×2.0 | +10% 稀有掉落 | 较高 |
| -3 | ×2.8 | +20% 稀有掉落 | 高 |
| -4 | ×3.5 | +30% + 特殊掉落 | 极高 |
| -5 ~ -9 | ×4.0（封顶） | +50% + 必出特殊掉落 | 险胜需顶配资源 |
| -10 及以上 | 不可战斗 | — | — |

> **数值策划注**：经验加成封顶 ×4.0，避免越级刷成主流玩法。配合每日打坐冷却与灵力消耗，**越级挑战在数学上不优于正常推进**，仅作为玩家"自我证明"的额外路径。

## 残魂阶段师父的战斗参与规则

> 参考来源：[《师徒系统设计》§4.2](09_师徒系统设计.md)

### 核心原则：零战斗干预

**自镇潮阁师父化为残魂后，师父在战斗系统中不具备任何数值效果。** 这是设计红线，不可突破。

| 维度 | 规则 | 原因 |
|------|------|------|
| 增伤 | ❌ 禁止 | 师父是叙事资产，不是战斗资产 |
| 减伤/护盾 | ❌ 禁止 | 同上 |
| 治疗/复活 | ❌ 禁止 | 残魂无灵力输出能力（世界观一致） |
| 战斗提示 | ✅ 允许（纯文字） | 识海低语提供敌人弱点提示，但**不触发任何战斗加成** |
| 濒死保护 | ❌ 禁止 | 第二幕起濒死文本改为"识海中传来叹息"，不再有师父实体接手 |

### 战斗中的识海低语呈现

战斗中师父的低语遵循以下规则：

- **首次遭遇新敌人**：师父可能触发一次低语（如"小心，它不是寻常水族"），纯叙事文本，不影响战斗数值。
- **反触发条件**：当玩家频繁请教（5分钟内>3次）时，师父不再回应——"识海中一片安静。他在等你自己想明白。"
- **Boss战中不打断**：Boss战的HP阶段转换文本优先级高于识海低语，师父低语仅在"退场"文本阶段出现。

### ⚠️ 设计警示

> **任何后续战斗系统设计中，禁止为师父残魂添加任何战斗向数值（增伤、减伤、复活等）。师父是叙事资产，不是战斗资产。违反此条应视为设计缺陷回退。**

## Boss战设计规范

### 规模分级

| 级别 | HP倍率（相对同级杂兵基线） | 阶段数 | 经验加成 | 文档示例 |
|------|----------------------|-------|---------|---------|
| **小Boss** | ×1.5 ~ ×1.8 | 2 阶段（HP 50% 转阶段） | ×1.5 | 水灵傀儡（80 HP，相对同级杂兵 ~50） |
| **中Boss** | ×2.0 ~ ×2.5 | 3 阶段（70%、35% 转阶段） | ×2.0 | 触须幻影（100 HP；非实体减伤实质 ~200 HP） |
| **大Boss** | ×3.0 ~ ×4.0 | 3~4 阶段，含场景互动 | ×2.5 | 江州终章预留（暂不实装）|

### HP 阈值阶段转换（以水灵傀儡为例）

| HP 阶段 | 行为变化 | 文字提示 |
|--------|---------|---------|
| 100% ~ 51% | 普攻为主，每3回合"涌泉击" | "傀儡之水身缓缓起伏，似在试探。" |
| **50% ~ 0%（狂涌）** | 攻击 +20%，每2回合"涌泉击" | "暗红灵核浮出胸口——它不是江水之子。" |

### Boss战操作小游戏

> **玩法策划准则**：每个 Boss 战必须包含 **1 个非战斗指令交互节点**，打破"出招—结算"的纯数值循环。

#### 示例：镇潮阁·水灵傀儡战（叩钟节奏）

战斗中段（傀儡 HP 60%），师父大声提示：
```
"徒儿——叩钟！按潮汐节拍！"
> [节奏小游戏开启]
> 屏幕显示提示：⬤ … ⬤ ⬤ … ⬤ … ⬤
> 输入：bell 1 (空) bell 2 bell 3 (空) bell 4 (空) bell 5
```

- **完美**：傀儡进入"静滞"2 回合（不能行动），玩家可自由输出。
- **普通**：静滞 1 回合。
- **失败**：傀儡狂暴，攻击额外 +30% 持续 2 回合。

> **关卡策划注**：节奏小游戏的提示符号应与场景内"潮汐声、钟声"叙事元素一致，避免突兀的"按键提示"破坏MUD沉浸感。

### Boss战叙事节点

每个 Boss 战必须包含以下 **3 个叙事节点**：

1. **入场**（战前 100~200 字）：环境压迫感铺垫；
2. **转阶段**（HP 阈值触发，80~150 字）：揭示 Boss 真容/动机；
3. **退场**（战后 150~300 字）：胜利文本 + 任务推进 / 师父或盟友评论。

### Boss战多道途适配要求（新增）

> 自闻道境第二幕（Lv.4+ 道途觉醒后），所有 Boss 级敌人须满足以下规范。详见 [《道途设计》](06_道途设计.md)。

1. **通关底线原则**：任何道途的玩家仅用**公共灵术**（灵气弹 + 聚气一击 + 普攻）配合消耗品也可通关。道途灵术是"效率工具"，不是"通关钥匙"。
2. **最契合标注**：每个 Boss 标注**一条最契合道途**（该道途面对此 Boss 通关回合数最短），用于引导而非限制。
3. **四路可通**：设计文档须包含四条道途各自的**策略简述**（推荐灵术 + 策略 + 预计回合 + 难度星级）。
4. **回合比例**：最契合道途 vs 最不契合道途的预期回合比 ≤ 1:2；纯公共灵术通关回合 ≤ 最契合道途的 2 倍。
5. **策略矩阵**：Lv.4+ 的 Boss 须附"多道途策略矩阵"表（格式参照敌人数据总表下方"多道途策略矩阵"一节）。

### 弹性经验奖励（自由度·第三层）

> **设计目标**：鼓励玩家挑战略高于自身的内容，避免反复刷低级怪，让"以弱胜强"成为有意义的选择。

**经验倍率公式**：

```
等级差 = 敌人等级 - 玩家等级
经验倍率：
  等级差 ≥ +2：×1.5 + 系统提示"以弱胜强——这场战斗你将铭记"
  等级差 = +1：×1.2
  等级差 = 0 ：×1.0
  等级差 = -1：×0.8
  等级差 ≤ -2：×0.5 + 系统提示"这对你已不是挑战"
```

**示例推演**：

| 玩家等级 | 敌人 | 敌人等级 | 基础经验 | 倍率 | 实际经验 |
|---------|------|---------|---------|------|---------|
| Lv.1 | 异化江鱼 | Lv.2 | 10 | ×1.2 | 12 |
| Lv.1 | 水纹石像 | Lv.3 | 25 | ×1.5 | 37 |
| Lv.3 | 异化江鱼 | Lv.2 | 10 | ×0.8 | 8 |
| Lv.4 | 异化江鱼 | Lv.2 | 10 | ×0.5 | 5 |

> **设计意图**：
> 1. 激进型玩家可以越级挑战高级敌人获得更多经验，加速成长——但需要承担更高的战斗风险
> 2. 反复刷低级怪的收益递减，引导玩家向前探索
> 3. ±2级是"甜蜜区"——大多数玩家自然处于推荐等级±1的范围
> 4. 经验倍率与挑战区间配合：水纹石像(Lv.3)对Lv.1玩家理论可打但极难，×1.5的经验是对勇气和技巧的奖励

## 数值平衡验证

> **数值策划论证流程**：每场关键战斗以"标准玩家"模拟（普攻取均值，灵术取均值，敌方均值，无暴击/未命中），列出回合推演。各验证结论见下文，完整逐回合推演存档于 [《数值验证附录》](11_数值验证附录.md)。

### 道途灵术与敌人特殊机制的交互规则

> 道途系统引入后，灵术分为**公共灵术**和**道途灵术**两类。道途灵术按灵气凝形方式分为三型，与敌人特殊机制的交互规则如下：

| 类型 | 代表灵术 | 非实体减伤 | 灵气压制光环 | 说明 |
|------|---------|-----------|------------|------|
| **纯灵术型** | 灵气弹、灵气涟漪、共振波、灵流锁 | 不受减伤（走 SpR 防御） | ×0.5（被完全压制） | 灵气能量直接释放 |
| **锋刃型** | 穿灵刺、破隙斩 | ×0.5（含物理成分） | ×0.75（灵气凝刃，介于纯灵术与物理之间） | 灵气凝形为物理刃 |
| **特化型** | **灵刃斩**（显式标注"无衰减"） | ×1.0（**无衰减**） | ×0.75（同锋刃型） | 专门设计用于对抗特殊机制 |
| **防御型** | 凝水盾、灵壁术、镇守 | — | 不受影响（护盾/嘲讽不算伤害） | 灵气凝形为结构 |

> 此分类确保：(1) 握锋在压制光环/非实体战中有独特优势；(2) 铸壁的防御灵术不被压制光环削弱；(3) 鸣钟在压制光环中受限最严重（纯灵术 ×0.5），但在群体战中天然强势。

### 验证 1：Lv.1 玩家 vs 异化江鱼 ×2（Lv.2）

**结论**：4 回合险胜，HP 余 24/50（48%，轻伤）。残血鱼设定确保"觉醒后力量爆发"的正反差体验。战后半血状态与阶段二必败形成情绪递进：绝望 → 觉醒 → 初胜 → 余悸犹在。

（完整推演见 [《数值验证附录》§验证1](11_数值验证附录.md#验证1)）


### 验证 2：Lv.3 玩家 vs 水纹石像（Lv.3，单只）

**结论**：5 回合击杀单只，HP 余 29/75（39%），灵力余 25/50。×2 同时遭遇时建议主动 flee 重组或借助场景古钟打断（关卡策划应在镇潮阁布置"古钟"互动物）。

（完整推演见 [《数值验证附录》§验证2](11_数值验证附录.md#验证2)）


### 验证 3：Lv.4 玩家 vs 水灵傀儡（Lv.4，小Boss·公共灵术）

> 此战发生在道途觉醒前，玩家仅有公共灵术（灵气弹 + 聚气一击 + 普攻）。

**结论**：纯公共灵术 7 回合通关。无叩钟时 HP 余 25%（重伤），叩钟完美时 HP 余 47%（中伤）。叩钟小游戏生存收益 = +20 HP，替代凝水盾成为"战斗中的防御节点"，与镇潮阁古钟场景叙事有机结合。

> **修订说明**：旧版验证使用了凝水盾（现归铸壁道途），本版修正为纯公共灵术。叩钟小游戏从"锦上添花"升级为"生存必需"。

（完整推演见 [《数值验证附录》§验证3](11_数值验证附录.md#验证3)）


### 验证 4：Lv.5 玩家 vs 灵气异化虫群（Lv.5，6只·保护陶翁）

> 虫群战发生在第二幕（玩家已选择道途），须**破坏灵气异化源**方可终结虫群补充。

**结论**：鸣钟（★最契合）3 回合清场+净化，HP 97%，陶翁安全（39/50）；握锋（最不契合）7 回合，HP 45%，陶翁仅剩 2.4 HP——"握锋不擅长保护战"的设计差异。

> **数值策划备注**：陶翁 HP 50 使四条道途体验差异化——鸣钟游刃有余，铸壁全程安全（灵壁术覆盖），观潮稳健（潮引术改变目标），握锋惊险过关。握锋玩家 T3 灵觉探查成功后获文字引导，暗示优先击破源头。

（完整推演见 [《数值验证附录》§验证4](11_数值验证附录.md#验证4)）


### 验证 4.5（新增）：Lv.5·握锋 vs 齿轮傀儡（Lv.5·引航台）

**结论**：5 回合，HP 余 57.1/110（52%，轻伤），灵力余 18/85。握锋通过"锋刃型灵术抗压制 ×0.75 + 破隙斩抓蓄力窗口"高效击杀，符合策略矩阵 5-6 回合预估。

> **对比**：鸣钟灵气涟漪被光环压制 ×0.5 需 7-8 回合；铸壁灵壁术隔绝光环后 8-9 回合但几乎无风险。握锋最快但吃重攻最疼——"高效但需精准操作"。

（完整推演见 [《数值验证附录》§验证4-5](11_数值验证附录.md#验证4-5)）


### 验证 5：Lv.6·握锋 vs 水兽触须幻影（Lv.6，中Boss·非实体）

**结论**：无闪避 7 回合（含 1 回合击退），HP 余 48/130（37%，中伤），灵力耗尽；闪避成功 5 回合，HP 余 80.8/130（62%）。灵刃斩 3 发后灵力枯竭需切换灵气弹/穿灵刺收尾——有意的资源瓶颈。

> **对比**：铸壁 9-11 回合（HP 60%+，几乎不死但极慢）；观潮 7-9 回合（HP 50%+，节奏从容）。四条道途体验差异在此 Boss 最为明显。

（完整推演见 [《数值验证附录》§验证5](11_数值验证附录.md#验证5)）

> *   **执行摘要**：完成《战斗数值设计》文档创建及道途系统整合修订。7 场关键战斗数学验证通过（含 2 场多道途对比验证）。

>     3. 境界压制 ±50%/-30% 封顶，10 级差以上转为"不可战胜"叙事；
>     4. 经数值复核回写：**虫群单只 HP 12→9**、**Boss 战追加"闪避反击"通用机制**；
>     5. **（道途修订）新增**：道途灵术三型分类（纯灵术/锋刃/特化）与敌人特殊机制交互规则表；

>     7. **（道途修订）新增**：Boss 战多道途适配规范（通关底线原则、回合比例 ≤1:2、策略矩阵模板）；
>     8. **（道途修订）修正**：验证 3 移除凝水盾（已归铸壁道途），改为纯公共灵术推演；验证 5 移除凝水盾+灵刃斩混用，改为纯握锋路线推演；新增验证 4.5（齿轮傀儡·握锋）。
> *   **内部争议与决策**：
>     - **数值 vs 剧情**：异化江鱼初战设计为"残血鱼vs满状态觉醒者"——主线任务已明确师父震荡波及鱼群降低其HP，因此以鱼HP 18/25（72%残血）计算验证；
>     - **玩法 vs 关卡**：触须幻影 9 回合超中Boss上限，玩法策划提出"闪避反击"，关卡策划确认现有云舟修葺台地形支持横向位移描写，通过；
>     - **文字 vs 系统**：所有 Boss 战必含 1 个非战斗指令交互节点，主策划拍板——这是"MUD 不能退化为数字对撞"的底线；
>     - **（道途修订）道途 vs 公平**：虫群保护战中握锋路线陶翁仅剩 2.4 HP——数值策划认可此为"有意的道途差异化体验"，非平衡缺陷。建议补充陶翁 HP=50 至 NPC 设定文档。
> *   **潜在风险提示**：
>     1. Lv.3 玩家面对水纹石像 ×2 同时遭遇时回合压力较大，**关卡策划需在镇潮阁布置古钟互动物兜底**，否则建议将一处遭遇改为单只；
>     2. 元素克制系数 ×1.5 较强，**剧情中火/雷外援的频率应受限**（建议每幕不超过 1 次明确克制助攻），避免玩家形成"等NPC放符"的依赖；
>     3. 越级挑战经验封顶 ×4.0 在江州篇足够，进入下一卷（见界境）需重新校准；
>     4. 所有玩家HP/MP数值严格沿用《玩家能力设计》标注值，未做任何修改，保持文档间一致性。
